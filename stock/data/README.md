# 股票分析数据目录

## 目录结构

```
data/
├── raw/                    # 原始数据（不改动的源数据）
│   ├── stocks/            # 股票基础数据
│   ├── daily/             # 日线行情原始数据
│   ├── fundflow/          # 资金流向原始数据
│   ├── market/            # 市场情绪原始数据
│   └── sectors/          # 板块原始数据
├── processed/             # 处理后的数据
│   ├── daily_calculated/ # 带技术指标的日线数据
│   ├── signals/          # 选股/择时信号
│   └── backtest/        # 回测结果
└── results/               # 分析结果输出
    ├── reports/          # 分析报告
    ├── charts/           # 图表
    └── exports/          # 导出文件
```

## 数据文件格式

### 日线数据格式 (CSV/Parquet)
```
trade_date,open,high,low,close,vol,amount
2026-02-13,10.52,10.68,10.45,10.60,1250000,13250000
```

### 资金流向数据格式
```
trade_date,ts_code,main_net_inflow,super_net_inflow,large_net_inflow
2026-02-13,000001,1250.5,850.3,-320.2
```

### 技术指标数据格式
```
trade_date,ma5,ma10,ma20,ma60,macd,k,d,rsi,boll_upper,boll_lower
2026-02-13,10.45,10.38,10.32,10.25,0.15,72.5,68.2,78.5,10.85,9.78
```

## 数据获取脚本

参考 `scripts/` 目录获取数据下载脚本。

## 数据更新频率

- **日线数据**: 每日收盘后更新
- **分钟数据**: 盘中实时更新
- **基本面数据**: 财报发布后更新
- **资金流向**: 每日收盘后更新

## 数据保留策略

- 日线数据：保留全部历史
- 分钟数据：保留最近3个月
- 临时文件：每周清理
- 备份：每月备份一次完整数据
