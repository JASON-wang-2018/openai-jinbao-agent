# 个股分析报告生成器

> 基于6步技术分析工作流的标准化报告输出

---

## 使用方法

```bash
# 分析指定股票
python stock/scripts/analyze_stock.py --code 002642

# 分析并保存报告
python stock/scripts/analyze_stock.py --code 002642 --save

# 对比分析
python stock/scripts/analyze_stock.py --code 002642 000001 600000

# 盘中快速扫描
python stock/scripts/analyze_stock.py --scan --limit 50
```

## 分析维度评分标准

### 均线结构（20分）
- 多头排列：20分
- 震荡整理：10分
- 空头排列：0分

### 量价关系（25分）
- 价涨量增：25分
- 价涨量缩：12分
- 价跌量缩：10分
- 价跌量增：15分

### K线形态（15分）
- 启动形态：15分
- 无明显形态：10分

### 主力行为（20分）
- 资金推升：20分
- 对倒/出货：8分
- 缩量上涨：12分
- 恐慌抛售/吸筹：15分

### 板块环境（20分）
- 热点+领涨：20分
- 板块数据待接入：10分

---

## 评分结果解读

| 分数 | 等级 | 操作建议 |
|------|------|---------|
| 80+ | 【强势主升】 | 积极参与，趋势持有 |
| 60-80 | 【可操作】 | 回调买入，择时操作 |
| 40-60 | 【观望】 | 等信号，不追高 |
| <40 | 【风险区】 | 回避或仅短线 |
