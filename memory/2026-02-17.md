## Excel 数据处理技能创建

### 背景
用户需求: 学习读取分析 Excel 数据，用于文本报告办公和项目管理（进度表等）

### 创建内容
**位置**: `/home/jason/.openclaw/workspace/skills/excel-data-processor/`

**核心文件**:
- `SKILL.md` (11KB) - 完整技能说明
- `excel_processor.py` (15KB) - 核心处理类
- `README.md` (7.4KB) - 使用指南
- `快速入门.md` (6.8KB) - 5分钟上手指南
- `examples/quick_start.py` (8.1KB) - 8个完整示例

### 核心功能

**ExcelProcessor 类**:
- 读取/写入 Excel（支持多 sheet）
- 数据清洗（空值、重复、格式）
- 合并多个文件
- 创建数据透视表
- 格式化 Excel（自动列宽、边框、标题样式）

**ProjectManager 类**:
- 创建进度跟踪表
- 创建资源分配表
- 创建风险登记册
- 生成项目状态报告（Markdown）
- 生成项目周报

### 技术栈

- openpyxl 3.1.5
- xlrd 2.0.2

### 使用方式
```python
from excel_processor import ExcelProcessor, ProjectManager

# 读取 Excel
processor = ExcelProcessor()
df = processor.read_excel('file.xlsx')

# 项目管理
pm = ProjectManager()
pm.create_progress_tracker('project.xlsx')
report = pm.generate_status_report('project.xlsx', 'report.md')
```
