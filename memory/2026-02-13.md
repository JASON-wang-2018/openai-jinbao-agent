## 两套核心分析模型

### 一、老股民警模型（stock/scripts/stock_analysis_old_trader.py）

**用途**：分析个股是否处于"主力可持续推升阶段"

**7步分析流程**：
1. 趋势与生命周期判断
2. 多维技术与行为证据分析
3. 主力行为模型判定
4. 80分主升评分
5. 走势概率推演
6. 策略与风控建议
7. 最终结构化输出

**使用命令**：
```bash
# 老股民警模型 - 个股分析
python stock/scripts/stock_analysis_old_trader.py --code 002642
```

### 二、市场大行情模型（stock/scripts/market_analysis.py）

**用途**：分析整个A股市场的资金动向、热点板块、机会与风险

**6维分析**：
1. 大盘趋势
2. 资金流向
3. 量能
4. 涨跌停
5. 板块轮动
6. 市场情绪

**评分体系**（100分制）：
- ≥80：强牛市
- 60-79：弱牛市
- 40-59：震荡市
- 20-39：弱熊市
- <20：强熊市

**使用命令**：
```bash
python stock/scripts/market_analysis.py
```

## 定时任务配置

### Cron 定时任务
- **每日8:00**：生成早盘复盘报告
- **每日21:00**：生成晚间复盘报告
- **每小时**：内存刷新检查
- **每30分钟**：上下文使用量检查

### 报告保存位置
```
stock/reports/
├── report_YYYY-MM-DD_am.txt   # 早盘报告
├── report_YYYY-MM-DD_pm.txt   # 晚间报告
├── market_YYYY-MM-DD.txt     # 市场分析报告
└── stock_代码_YYYY-MM-DD.txt # 个股分析报告
```
