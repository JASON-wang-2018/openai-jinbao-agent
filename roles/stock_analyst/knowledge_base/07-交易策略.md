### 2.1 核心逻辑

> 趋势是你的朋友
> 上涨的股票还会继续上涨，直到有一天不再上涨

### 2.2 均线策略

```python
# 双均线策略
def 均线趋势(df, short=5, long=20):
    ma_short = df['close'].rolling(short).mean()
    ma_long = df['close'].rolling(long).mean()
    
    # 金叉买入
    if ma_short.iloc[-1] > ma_long.iloc[-1] and \
       ma_short.iloc[-2] <= ma_long.iloc[-2]:
        return "买入"
    
    # 死叉卖出
    if ma_short.iloc[-1] < ma_long.iloc[-1] and \
       ma_short.iloc[-2] >= ma_long.iloc[-2]:
        return "卖出"
    
    return "持有"
```

### 2.3 突破策略

```python
# 突破新高策略
def 突破新高选股(df, lookback=20, pct=0.02):
    # 突破近N日高点
    high = df['high'].rolling(lookback).max()
    
    # 突破确认（+2%过滤假突破）
    cond = df['close'] > high.shift(1) * (1 + pct)
    return cond.iloc[-1]
```

## 三、价值投资策略

### 3.1 核心逻辑

> 以合理的价格买入优秀的企业
> 长期持有，享受复利

### 3.2 选股标准

```python
# 价值股筛选
def 价值股筛选(stocks):
    candidates = []
    for stock in stocks:
        if stock.pe < 15 and stock.pe > 0:  # 低PE
            if stock.pb < 2:                 # 低PB
                if stock.dividend_yield > 2: # 高股息
                    if stock.roe > 10:        # 高ROE
                        candidates.append(stock)
    return candidates
```

### 3.3 买入策略

| 策略 | 方法 |
|------|------|
| 格雷厄姆 | PE<15, PB<1.5 |
| 施洛斯 | 低于净流动资产 |
| 巴菲特 | 优秀生意+合理价格 |

### 3.4 卖出策略

1. **基本面恶化**
2. **估值过高**
3. **发现更好的标的**
4. **系统性风险**

## 五、量化策略框架

### 5.1 策略开发流程

```
1. 策略想法 → 2. 数据验证 → 3. 回测检验 → 4. 实盘优化
```

### 5.2 回测框架

```python
def backtest(strategy, data, start_date, end_date):
    signals = strategy.generate_signals(data)
    returns = data['close'].pct_change()
    
    # 计算收益
    portfolio = (1 + returns * signals).cumprod()
    
    # 计算指标
    total_return = portfolio.iloc[-1] - 1
    max_drawdown = (portfolio / portfolio.cummax() - 1).min()
    sharpe_ratio = returns.mean() / returns.std() * np.sqrt(252)
    
    return {
        'total_return': total_return,
        'max_drawdown': max_drawdown,
        'sharpe_ratio': sharpe_ratio
    }
```

## 六、策略组合

### 6.1 策略配置

| 资金比例 | 策略 | 预期收益 | 最大回撤 |
|---------|------|---------|---------|
| 40% | 趋势跟踪 | 15-25% | 15-20% |
| 30% | 价值投资 | 10-15% | 10-15% |
| 20% | 短线博弈 | 20-40% | 20-30% |
| 10% | 现金备用 | 2-3% | 0% |

### 6.2 策略相关性

- 策略之间相关性 < 0.5
- 避免同质化
- 分散风格风险

### 6.3 动态平衡

- 定期再平衡（季度/半年）
- 动态调整策略权重
- 根据市场环境切换
