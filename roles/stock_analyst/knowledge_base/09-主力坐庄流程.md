## 二、各阶段特征详解

### 1. 吸筹建仓阶段

**时间**：3-6个月甚至更长

**目的**：在低位收集足够多的筹码

**特征**：
- 成交量：温和放大，持续性放量
- 股价：区间震荡，涨跌无量
- 分时图：早盘快速拉升后全天横盘
- K线：小阴小阳，振幅收窄
- 均线：开始走平，方向模糊

**主力行为**：
- 打压吸筹：利空消息配合，砸盘收集
- 拉高吸筹：快速脱离成本区
- 横盘吸筹：长期横盘，消磨散户耐心

**识别技巧**：
- 利好不涨，利空不跌
- 频繁出现单针探底
- 成交量放大但股价波动小

---

### 2. 洗盘整理阶段

**时间**：1-3个月

**目的**：清洗浮筹，提高市场成本

**特征**：
- 成交量：回调时缩量
- 股价：回踩重要支撑不破
- 分时图：无量下跌，承接有力
- K线：出现上下影线

**洗盘方式**：
- 打压洗盘：快速急跌，制造恐慌
- 横盘洗盘：长时间横盘震荡
- 震荡洗盘：宽幅震荡

**识别技巧**：
- 缩量回调不破位
- 关键位置出现支撑
- 盘整后快速拉起

---

### 3. 试盘阶段

**时间**：几天到几周

**目的**：测试上方抛压和跟风盘

**特征**：
- 成交量：突然放量
- 股价：冲高回落或突破测试
- 分时图：快速拉升后回落

**试盘类型**：
- 向上试盘：测试压力位
- 向下试盘：测试支撑位

**识别技巧**：
- 突然放量异动
- 冲高后快速回落
- 收盘收上影线

---

### 4. 拉升阶段

**时间**：主升浪1-3个月

**目的**：快速脱离成本区，兑现利润

**特征**：
- 成交量：持续放大，换手率升高
- 股价：连续上涨，流畅的上涨趋势
- 分时图：早盘快速涨停或稳步推升
- K线：大阳线居多，常出现涨停板

**拉升方式**：
- 快速拉升：连续涨停板
- 稳步推升：沿着均线上涨
- 台阶式拉升：涨一平台整理再拉升

**识别技巧**：
- 均线多头排列
- 成交量持续放大
- 出现板块联动

---

### 5. 高位派发阶段

**时间**：1-2个月

**目的**：在高位出货，兑现利润

**特征**：
- 成交量：放大但滞涨
- 股价：宽幅震荡
- 分时图：脉冲式拉升
- K线：出现吊颈线、乌云盖顶

**出货方式**：
- 震荡出货：高位宽幅震荡
- 利好出货：配合利好消息出货
- 跌停出货：恐慌性出货

**识别技巧**：
- 利好频传但股价滞涨
- 多次冲击高点失败
- 分时图无量上涨

---

### 6. 下跌出清阶段

**时间**：1-3个月

**目的**：清仓出逃，股价回归

**特征**：
- 成交量：萎缩
- 股价：持续下跌
- 均线：空头排列

## 四、主力行为识别模型

### 吸筹建仓识别

```python
def 识别吸筹(df):
    """识别吸筹建仓阶段"""
    特征 = []
    
    # 特征1：成交量温和放大
    vol_ma20 = df['volume'].rolling(20).mean()
    if df['volume'].iloc[-1] > vol_ma20.iloc[-1] * 1.2:
        特征.append("量能温和放大")
    
    # 特征2：股价波动收窄
    volatility = (df['high'] - df['low']) / df['close']
    if volatility.iloc[-1] < volatility.rolling(20).mean().iloc[-1] * 0.8:
        特征.append("波动收窄")
    
    # 特征3：均线走平
    ma5, ma20, ma60 = 计算均线(df)
    if abs(ma5 - ma20) < 0.02 * ma20:
        特征.append("均线走平")
    
    return 特征
```

### 洗盘识别

```python
def 识别洗盘(df):
    """识别洗盘阶段"""
    # 回调缩量
    vol_ma20 = df['volume'].rolling(20).mean()
    if df['volume'].iloc[-1] < vol_ma20.iloc[-1] * 0.7:
        return "洗盘特征：缩量回调"
    
    # 不破关键位
    ma20 = df['close'].rolling(20).mean().iloc[-1]
    if df['close'].iloc[-1] > ma20 * 0.98:
        return "洗盘特征：MA20有支撑"
    
    return None
```

### 拉升识别

```python
def 识别拉升(df):
    """识别拉升阶段"""
    # 量价齐升
    vol_ratio = df['volume'].iloc[-1] / df['volume'].rolling(20).mean().iloc[-1]
    price_change = (df['close'].iloc[-1] - df['close'].iloc[-5]) / df['close'].iloc[-5]
    
    if vol_ratio > 1.5 and price_change > 0.1:
        return "拉升特征：量价齐升"
    
    # 均线多头
    ma5, ma20 = 计算均线(df)
    if ma5 > ma20 > ma20 * 1.02:
        return "拉升特征：均线多头"
    
    return None
```

## 五、经典坐庄案例

### 案例1：强势庄股

**特征**：
- 连续涨停板
- 成交量极度放大
- 利好消息配合

**操作**：
- 第一个涨停板介入
- 持有到放巨量滞涨

### 案例2：温和型庄股

**特征**：
- 稳步推升
- 成交量温和
- 洗盘规律

**操作**：
- 沿5日均线低吸
- 跌破10日均线止损

### 案例3：长线庄股

**特征**：
- 长期横盘
- 吸筹时间长达半年以上
- 启动前挖坑

**操作**：
- 跟踪成交量变化
- 突破平台后介入

## 六、应对策略

### 吸筹阶段

**策略**：
- 逢低分批建仓
- 不追高
- 保持耐心

**仓位**：30%

### 洗盘阶段

**策略**：
- 持股不动
- 回调加仓
- 不被洗出

**仓位**：50%

### 拉升阶段

**策略**：
- 顺势而为
- 沿均线持有
- 不轻易下车

**仓位**：70%

### 派发阶段

**策略**：
- 分批减仓
- 不追高买入
- 准备离场

**仓位**：30%

## 八、常见骗线手法

### 1. 假突破

**特征**：
- 早盘快速冲高
- 午后回落
- 收盘跌破突破位

**应对**：
- 等待收盘确认
- 不追突破

### 2. 假摔

**特征**：
- 盘中快速下跌
- 尾盘拉回
- 成交量不放大

**应对**：
- 不恐慌抛售
- 可能是加仓机会

### 3. 假突破真出货

**特征**：
- 突破后快速回落
- 成交量异常放大
- 多次冲击失败

**应对**：
- 减仓应对
- 不追高
