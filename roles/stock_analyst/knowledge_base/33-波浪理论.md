# 波浪理论

> 艾略特波浪理论，技术分析的经典之作。
> 以"五升三降"为核心，描述市场循环规律。

---

## 一、波浪理论基础

### 1.1 核心思想

```
波浪理论核心：
├── 市场价格呈波浪式波动
├── 上涨5浪，下跌3浪
├── 浪与浪之间有比例关系
└── 形态、比例、时间三要素
```

### 1.2 基本原则

| 原则 | 说明 |
|------|------|
| 浪3不能最短 | 浪3通常是最长的推动浪 |
| 浪2不破浪1起点 | 浪2回调不能跌破浪1起点 |
| 浪4不破浪1顶点 | 浪4回调不能进入浪1区间 |
| 交替原则 | 浪2与浪4形态交替 |

### 1.3 三要素

| 要素 | 说明 | 重要性 |
|------|------|--------|
| **形态** | 波浪的结构 | ⭐⭐⭐⭐⭐ |
| **比例** | 浪与浪的比例关系 | ⭐⭐⭐ |
| **时间** | 浪的持续时间 | ⭐⭐ |

## 二、推动浪（5浪上涨）

### 2.1 五浪结构

```
推动浪结构：

    1浪      2浪      3浪      4浪      5浪
     ↑       ↓        ↑        ↓        ↑
    / \     / \      / \      / \      / \
   /   \   /   \    /   \    /   \    /   \
--/-----\_/-----\__/-----\__/-----\__/-----\__/-----> 时间

  起点                                            终点
```

### 2.2 各浪特征

| 浪 | 名称 | 特征 | 幅度 | 成交量 |
|----|------|------|------|--------|
| **1浪** | 启动浪 | 试探性上涨，成交量温和放大 | 1浪起点~1浪顶 | 温和放大 |
| **2浪** | 回调浪 | 回调幅度大，通常回吐1浪的50-78.6% | 1浪的50-78.6% | 缩量 |
| **3浪** | 主升浪 | 最强劲，涨幅最大，通常是1浪的1.618-2.618倍 | 1浪的1.618-2.618倍 | 放量 |
| **4浪** | 回调浪 | 回调幅度小，通常回吐3浪的38.2% | 3浪的38.2% | 缩量 |
| **5浪** | 冲刺浪 | 涨幅小于3浪，通常是1浪的1-1.618倍 | 1浪的1-1.618倍 | 放量或背离 |

### 2.3 1浪特征

```
1浪特征：
├── 底部反转信号
├── 涨幅一般较小（10-30%）
├── 成交量温和放大
├── 均线开始走平/向上
├── 多数人认为是反弹
└── 突破下降趋势线
```

### 2.4 2浪特征

```
2浪特征：
├── 回调幅度大（50-78.6%）
├── 成交量明显萎缩
├── 恐慌性抛售
├── 创新低可能性小
├── 多数人认为熊市未结束
└── 形成底部形态
```

### 2.5 3浪特征

```
3浪特征：
├── 最强劲，涨幅最大
├── 涨幅通常是1浪的1.618-2.618倍
├── 成交量显著放大
├── 均线多头排列
├── 突破前期高点
├── 吸引场外资金
├── 市场情绪高涨
└── 涨停板增多
```

### 2.6 4浪特征

```
4浪特征：
├── 回调幅度小（38.2%）
├── 成交量萎缩
├── 横向整理
├── 三角形整理居多
├── 洗盘整理
├── 主力调仓换股
├── 为5浪蓄势
└── 不破1浪顶点
```

### 2.7 5浪特征

```
5浪特征：
├── 涨幅小于3浪
├── 涨幅通常是1浪的1-1.618倍
├── 成交量可能放大或背离
├── 均线多头排列
├── 鸡犬升天
├── 利好不断
├── 散户追涨
├── 风险累积
└── 筑顶信号
```

## 三、调整浪（3浪下跌）

### 3.1 三浪结构

```
调整浪结构：

       A浪         B浪         C浪
        ↓           ↑           ↓
       / \         / \         / \
      /   \       /   \       /   \
-----/-----\_____/-----\_____/-----\----- 时间

    起点                          终点
```

### 3.2 调整浪类型

| 类型 | 结构 | 特征 | 幅度 |
|------|------|------|------|
| **锯齿形** | 5-3-5 | 简单、快速 | 回调深 |
| **平台形** | 3-3-5 | 复杂、横盘 | 回调浅 |
| **三角形** | 3-3-3-3-3 | 收敛、整理 | 回调浅 |
| **双三** | 组合 | 复杂 | 回调中 |
| **三三** | 组合 | 最复杂 | 回调浅 |

### 3.3 锯齿形调整

```
锯齿形（5-3-5）：

       A              C
      / \           / \
     /   \    B    /   \
____/_____\_______/_____\____ 时间

特征：
├── A浪5浪结构
├── B浪3浪结构
├── C浪5浪结构
├── 回调幅度大（61.8-80%）
└── 快速下跌
```

### 3.4 平台形调整

```
平台形（3-3-5）：

       A              C
      / \           / \
     /   \    B    /   \
____/_____\_______/_____\____ 时间

特征：
├── A浪3浪结构
├── B浪3浪结构
├── C浪5浪结构
├── 回调幅度浅（38.2%）
└── 横向整理
```

### 3.5 三角形调整

```
三角形（3-3-3-3-3）：

       A
      / \
     /   \   D
    /     \ / \
   /   B   C   \ E
__/_____________\______ 时间

特征：
├── 5个子浪都是3浪结构
├── 回调幅度浅（50%）
├── 收敛整理
├── 方向选择
└── 突破后加速
```

## 四、浪的比例关系

### 4.1 推动浪比例

| 比例 | 含义 |
|------|------|
| 浪3 = 浪1 × 1.618 | 常见比例 |
| 浪3 = 浪1 × 2.618 | 强势比例 |
| 浪5 = 浪1 × 1.0 | 等长比例 |
| 浪5 = 浪1 × 0.618 | 失败5浪 |
| 浪5 = 浪1-3 × 0.618 | 菲波那契比例 |

### 4.2 回调浪比例

| 回调浪 | 回调幅度 | 意义 |
|--------|----------|------|
| 浪2回调浪1 | 50-61.8% | 正常回调 |
| 浪2回调浪1 | 78.6% | 深度回调 |
| 浪4回调浪3 | 38.2% | 浅度回调 |
| 浪4回调浪3 | 50% | 正常回调 |
| A浪回调5浪 | 38.2-50% | 正常回调 |
| A浪回调5浪 | 61.8-80% | 深度回调 |

### 4.3 黄金分割

```
常用黄金分割比例：
```

## 五、波浪的级别

### 5.1 时间级别

| 级别 | 周期 | 浪的持续时间 | 适合操作 |
|------|------|--------------|----------|
| 超大循环 | 世纪 | 数十年 | 投资 |
| 大循环 | 10-20年 | 数年 | 投资 |
| 中循环 | 1-3年 | 1-2年 | 长线 |
| 基础循环 | 3月-1年 | 数月 | 中线 |
| 中型浪 | 1-3月 | 数周-数月 | 中短线 |
| 小型浪 | 1-4周 | 1-3周 | 短线 |
| 微浪 | 1-7天 | 1-5天 | 超短 |
| 次微浪 | <1天 | 1-4小时 | 日内 |

### 5.2 级别对应

```
1个超大循环 = 2个大循环
1个大循环 = 8个中循环
1个中循环 = 3个基础循环
1个基础循环 = 5个中型浪
1个中型浪 = 8个小型浪
```

## 六、波浪理论的应用

### 6.1 数浪规则

```
数浪基本原则：
1. 浪3不能是最短的推动浪
2. 浪2不能跌破浪1起点
3. 浪4不能进入浪1区间
4. 浪1、3、5不能同时重叠
5. 交替原则：浪2与浪4形态交替

数浪方法：
1. 从左到右找起点
2. 识别推动浪和调整浪
3. 确认浪与浪的比例
4. 判断当前所处位置
5. 预测未来走势
```

### 6.2 浪的识别

```
推动浪识别：
├── 5波结构
├── 高点不断抬高
├── 低点不断抬高
└── 成交量配合

调整浪识别：
├── 3波结构
├── 高点降低
├── 低点降低
└── 成交量萎缩
```

### 6.3 预测未来

```
预测方法：
1. 根据比例推算目标位
2. 根据形态判断转折点
3. 根据时间判断周期
4. 根据背驰判断终结

预测公式：
├── 浪3目标 = 浪1起点 + (浪1幅度 × 1.618)
├── 浪5目标 = 浪1起点 + (浪1+浪3幅度 × 0.618)
├── A浪目标 = 浪5终点 - (浪5幅度 × 0.618)
└── C浪目标 = A浪起点 - (A浪幅度 × 1.618)
```
