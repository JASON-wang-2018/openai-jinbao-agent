## 一、牛熊周期规律

### 周期划分

```
┌─────────────────────────────────────────────────────────────────┐
│                     A股典型牛熊周期                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  底部吸筹期 → 牛市启动 → 主升浪 → 顶部派发 → 熊市下跌 → 底部吸筹  │
│     ↓                                    ↓                       │
│   1-6个月        3-6个月      6-12个月    3-6个月             │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 各阶段特征

| 阶段 | 持续时间 | 特征 | 操作策略 |
|------|---------|------|---------|
| 底部吸筹 | 1-6月 | 地量、阴跌、信心丧失 | 逐步建仓 |
| 牛市启动 | 3-6月 | 放量、板块轮动 | 积极参与 |
| 主升浪 | 6-12月 | 趋势上涨、热点扩散 | 持股待涨 |
| 顶部派发 | 3-6月 | 滞涨、天量、政策收紧 | 逐步减仓 |
| 熊市下跌 | 3-6月 | 阴跌、反弹无力 | 轻仓/空仓 |
| 恐慌探底 | 1-3月 | 放量暴跌、绝望 | 准备抄底 |

### 行业轮动规律

**经济周期视角**：

```
经济复苏 → 强周期股领先 → 煤炭、有色、化工、地产
     ↓
经济扩张 → 消费股发力 → 医药、白酒、食品、汽车
     ↓
经济过热 → 资源股高峰 → 煤炭、有色、石油
     ↓
经济滞胀 → 防御股避险 → 医药、公用事业
     ↓
经济衰退 → 现金为王 → 债券、货币基金
```

## 三、热点板块轮动模式

### 板块轮动节奏

| 轮动顺序 | 板块 | 持续时间 | 特征 |
|---------|------|---------|------|
| 1 | 券商 | 1-2周 | 牛市旗手，率先启动 |
| 2 | 保险 | 1周 | 金融板块联动 |
| 3 | 银行 | 1周 | 估值修复 |
| 4 | 地产 | 2周 | 政策驱动 |
| 5 | 周期 | 2-4周 | 经济复苏逻辑 |
| 6 | 消费 | 4周 | 业绩验证 |
| 7 | 医药 | 4周 | 防御属性 |
| 8 | 科技 | 2-8周 | 成长逻辑 |
| 9 | 题材 | 1-4周 | 事件驱动 |

### 板块联动模式

**横向联动（同属性）**：

```
同一热点 → 相关题材扩散
AI → 大模型 → 算力 → 应用 → 数据要素
新能源 → 锂电池 → 光伏 → 储能 → 整车
```

**纵向联动（产业链）**：

```
上游资源 → 中游制造 → 下游应用
锂矿 → 锂电池 → 新能源车
硅料 → 硅片 → 组件 → 光伏电站
```

## 四、题材炒作生命周期

### 题材阶段划分

```
启动期 → 发酵期 → 高潮期 → 分化期 → 退潮期 → 冰点期
   ↓         ↓         ↓        ↓        ↓        ↓
 1-2天     3-5天     1-2周    3-5天    1-2周    1-4周
```

### 各阶段特征

| 阶段 | 特征 | 操作策略 |
|------|------|---------|
| 启动期 | 板块异动，1-2只涨停 | 第一时间上车 |
| 发酵期 | 板块扩散，多只涨停 | 找板块龙一 |
| 高潮期 | 批量涨停，扩散效应 | 持股待涨，减仓 |
| 分化期 | 淘汰赛开始 | 去弱留强 |
| 退潮期 | 亏钱效应 | 止损离场 |
| 冰点期 | 板块沉寂 | 等待新周期 |

### 题材级别判断

| 级别 | 持续时间 | 涨幅空间 | 案例 |
|------|---------|---------|------|
| 超级题材 | 3-6月 | 3-5倍 | 新能源车、AI |
| 中级题材 | 1-3月 | 1-2倍 | 元宇宙、固态电池 |
| 短线题材 | 1-4周 | 30-80% | 事件驱动 |

### 判断依据

1. **政策力度** - 国家级 > 部委级 > 地方级
2. **产业想象空间** - 万亿 > 千亿 > 百亿
3. **业绩兑现度** - 立即 > 远期
4. **市场认可度** - 资金追捧程度

## 五、板块轮动实战技巧

### 1. 捕捉板块轮动

**信号**：
- 成交量放大
- 板块涨幅领先
- 出现板块涨停潮
- 资金持续流入

### 2. 判断板块持续性

| 信号 | 持续性强 | 持续性弱 |
|------|---------|---------|
| 涨停数量 | 多（>10只） | 少（<5只） |
| 成交量 | 持续放大 | 一日游 |
| 跟风效应 | 扩散 | 独自上涨 |
| 消息面 | 持续催化 | 一次过 |

### 3. 板块切换节奏

```
信号1：券商板块异动 → 牛市预期
信号2：成交量持续放大 → 增量资金入场
信号3：低位板块启动 → 板块轮动开始
信号4：高位股补跌 → 风险偏好下降
```

## 七、成交量与板块轮动

### 放量信号

| 放量类型 | 含义 | 操作 |
|---------|------|------|

### 量价配合

| 板块表现 | 量能 | 含义 |
|---------|------|------|
| 板块领涨 | 放量 | 资金追捧，延续 |
| 板块领涨 | 缩量 | 诱多，谨慎 |
| 板块跟涨 | 放量 | 资金关注，跟随 |
| 板块跟涨 | 缩量 | 一日游，不跟 |

## 八、板块轮动策略

### 策略1：跟热点

```
选择当日领涨板块 → 找板块龙一 → 回调买入 → 板块退潮卖出
```

### 策略2：潜伏低吸

```
找即将轮动的板块 → 提前埋伏 → 放量启动卖出 → 切换下一个
```

### 策略3：持股待涨

```
选定主线板块 → 持股不动 → 板块高潮减仓 → 去弱留强
```
