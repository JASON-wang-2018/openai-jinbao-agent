## 三、业绩类题材

### 3.1 业绩预增

```
特征：
├── 公告前小幅上涨
├── 公告后可能高开低走
└── 超预期才能连板

操作策略：
├── 潜伏：公告前1-2周介入
├── 追涨：公告后竞价超预期
└── 避险：利好出尽跑路
```

### 3.2 扭亏为盈

```
特征：
├── 业绩拐点
├── 估值重估
└── 可能摘帽

操作策略：
├── 关注亏损减少的公司
├── 关注主营改善的公司
└── 关注摘帽预期
```

### 3.3 摘帽行情

```
特征：
├── 摘帽前涨幅大
├── 摘帽后可能利好出尽
└── ST板块弹性大

操作策略：
├── 潜伏：摘帽前1个月
├── 追涨：摘帽当日抢筹
└── 卖出：摘帽后冲高跑
```

## 八、题材生命周期

### 8.1 五阶段模型

```
启动期 → 发酵期 → 加速期 → 分化期 → 退潮期
   ↓          ↓         ↓         ↓        ↓
  1-3天     3-7天     5-15天    3-7天    1-3天
```

### 8.2 各阶段特征

| 阶段 | 涨幅 | 成交量 | 涨停数 | 板块强度 |
|------|------|--------|--------|----------|
| 启动期 | 10-30% | 温和放大 | 2-5家 | 弱 |
| 发酵期 | 30-50% | 持续放量 | 5-20家 | 中 |
| 加速期 | 50-100% | 天量 | >20家 | 强 |
| 分化期 | 震荡 | 巨量 | 10-20家 | 弱 |
| 退潮期 | -20-50% | 缩量 | <10家 | 弱 |

### 8.3 操作策略

| 阶段 | 仓位 | 策略 | 标的选择 |
|------|------|------|----------|
| 启动期 | 30% | 轻仓试错 | 板块龙头 |
| 发酵期 | 50% | 积极参与 | 板块龙头+中军 |
| 加速期 | 70% | 重仓持有 | 板块龙头 |
| 分化期 | 30% | 去弱留强 | 最强龙头 |
| 退潮期 | 0% | 空仓观望 | 不操作 |

## 九、题材选股法则

### 9.1 选股优先级

```
T1：板块龙头（身位领先、连板最高）
T2：板块中军（成交额最大、机构青睐）
T3：补涨龙（板块翻倍后出现）
T4：跟风股（板块炒作末期）
```

### 9.2 题材质量评估

| 评估项 | 高分标准 | 权重 |
|--------|----------|------|
| 政策力度 | 国家战略级 | 30% |
| 行业空间 | 万亿级以上 | 20% |
| 业绩兑现 | 有业绩支撑 | 20% |
| 市场认同 | 资金认可 | 15% |
| 持续时间 | 1年以上 | 15% |

## 十、题材炒作规律

### 10.1 时间规律

```
1. 新题材启动后一般炒作3-7天
2. 大题材炒作1-3个月
3. 超级题材炒作1年以上
4. 题材落地后一般炒作1-2周
```

### 10.2 空间规律

```
1. 小题材涨幅50-100%
2. 中题材涨幅100-300%
3. 大题材涨幅300-1000%
4. 超级题材涨幅1000%以上
```

### 10.3 节奏规律

```
第一波：龙头领涨（涨幅最大）
第二波：板块轮涨（涨幅次之）
第三波：补涨炒作（涨幅最小）
```

## 十二、题材风险识别

### 12.1 风险类型

| 风险 | 表现 | 应对 |
|------|------|------|
| 利好出尽 | 业绩公布后下跌 | 提前埋伏，冲高跑 |
| 监管风险 | 交易所问询/停牌 | 关注监管动态 |
| 竞争加剧 | 同行扩产、价格战 | 关注行业格局 |
| 预期落空 | 政策不及预期 | 及时止损 |

### 12.2 退潮信号

```
退潮信号：
├── 龙头股炸板率高
├── 跟风股批量跌停
├── 板块内部分化
├── 成交量萎缩
└── 媒体热度下降
```

## 十六、学习总结

```
题材分类要点：
1. 政策级别决定行情大小
2. 行业空间决定涨幅空间
3. 业绩兑现决定持续时间
4. 资金认可决定炒作力度
5. 生命周期决定操作节奏

题材操作要点：
1. 选大题材，不炒小题材
2. 选龙头，不炒杂毛
3. 买在启动，不追高位
4. 卖在退潮，不格局
5. 设好止损，不死扛
```
