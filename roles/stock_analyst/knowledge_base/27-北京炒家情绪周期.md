# 北京炒家情绪周期理论

> 北京炒家，80后，北京人氏。
> 以"情绪周期理论"著称，操作风格稳健。
> 善于把握市场情绪，捕捉情绪转折点。

---

## 一、情绪周期核心概念

### 1.1 什么是情绪周期

```
情绪周期 = 市场情绪的轮回

冰点 → 复苏 → 发酵 → 高潮 → 分化 → 退潮 → 冰点

每一个轮回，就是一个周期
每一个周期，都有规律可循
把握周期，就是把握赚钱效应
```

### 1.2 情绪周期的本质

```
情绪 = 资金情绪 + 散户情绪 + 媒体情绪

资金情绪：大资金的买入卖出
散户情绪：散户的追涨杀跌
媒体情绪：新闻报道的乐观悲观

三者共振 → 情绪极值
三者背离 → 情绪修复
```

## 二、六阶段详解

### 2.1 冰点期

```
特征：
├── 涨停家数 < 30
├── 连板高度 < 3
├── 炸板率 > 60%
├── 跌停家数 > 20
├── 成交额 < 1.5万亿
├── 涨跌比 < 0.5
└── 市场情绪：恐慌、绝望

市场表现：
├── 题材全部熄火
├── 龙头补跌
├── 闪崩股增多
├── 成交量萎缩
└── 利好不涨

操作策略：
├── 空仓等待
├── 不抄底
├── 不抢反弹
└── 等待冰点转折

冰点信号：
├── 连续3天冰点
├── 出现恐慌性杀跌
├── 成交量创近期新低
└── 跌停家数增加
```

### 2.2 复苏期

```
特征：
├── 涨停家数 30-50
├── 连板高度 3-5
├── 炸板率 40-60%
├── 跌停家数 < 10
├── 成交额 1.5-1.8万亿
├── 涨跌比 0.5-1
└── 市场情绪：犹豫、试探

市场表现：
├── 新题材出现
├── 试错资金进场
├── 连板股增多
├── 炸板率下降
└── 成交量放大

操作策略：
├── 轻仓试错（20%）
├── 只做新题材首板
├── 止损要果断
└── 不格局

复苏信号：
├── 新题材首板增多
├── 昨日涨停溢价
├── 连板高度突破
└── 板块效应出现
```

### 2.3 发酵期

```
特征：
├── 涨停家数 50-100
├── 连板高度 5-7
├── 炸板率 30-40%
├── 跌停家数 < 5
├── 成交额 > 1.8万亿
├── 涨跌比 1-1.5
└── 市场情绪：乐观、积极

市场表现：
├── 主线板块明确
├── 龙头股持续涨停
├── 跟风股批量涨停
├── 板块效应明显
├── 赚钱效应扩散
└── 场外资金入场

操作策略：
├── 积极参与（50%）
├── 只做主线板块
├── 只做板块龙头
├── 适当格局
└── 分批建仓

发酵信号：
├── 主线板块持续
├── 龙头股超预期
├── 板块内连板增多
├── 成交量持续放大
└── 媒体开始报道
```

### 2.4 高潮期

```
特征：
├── 涨停家数 > 100
├── 连板高度 > 7
├── 炸板率 < 30%
├── 跌停家数 < 2
├── 成交额 > 2万亿
├── 涨跌比 > 2
└── 市场情绪：亢奋、全民炒股

市场表现：
├── 个股普涨
├── 龙头加速
├── 补涨股增多
├── 新题材层出不穷
├── 成交量创天量
├── 券商开户激增
└── 媒体大幅报道

操作策略：
├── 边打边撤（50%）
├── 去弱留强
├── 锁定利润
├── 不开新仓
└── 准备减仓

高潮信号：
├── 龙头加速涨停
├── 补涨股全部涨停
├── 成交量创天量
├── 出现一字板潮
├── 菜市场都在讨论股票
```

### 2.5 分化期

```
特征：
├── 涨停家数下降（100→50）
├── 连板高度下降（7→5）
├── 炸板率上升（30%→50%）
├── 涨跌比下降（2→1）
├── 成交额下降
└── 市场情绪：分歧、犹豫

市场表现：
├── 板块内部分化
├── 跟风股先跌
├── 龙头股滞涨
├── 炸板股增多
├── 亏钱效应出现
└── 资金开始出逃

操作策略：
├── 降低仓位（30%）
├── 去弱留强
├── 锁定利润
├── 不追涨
└── 准备撤退

分化信号：
├── 龙头股炸板
├── 跟风股跌停
├── 板块内部分化
├── 炸板率上升
└── 亏钱效应扩散
```

### 2.6 退潮期

```
特征：
├── 涨停家数 < 50
├── 连板高度 < 3
├── 炸板率 > 50%
├── 跌停家数 > 10
├── 成交额 < 1.5万亿
├── 涨跌比 < 0.5
└── 市场情绪：悲观、绝望

市场表现：
├── 题材全部熄火
├── 龙头股补跌
├── 闪崩股增多
├── 成交量萎缩
├── 亏钱效应扩散
└── 资金观望

操作策略：
├── 降低仓位（0-20%）
├── 止损要果断
├── 不抄底
├── 不抢反弹
└── 等待冰点

退潮信号：
├── 龙头股跌停
├── 板块集体下跌
├── 炸板股次日低开
├── 跌停股增多
└── 成交量萎缩
```

## 三、情绪周期量化

### 3.1 情绪指标表

| 指标 | 冰点 | 复苏 | 发酵 | 高潮 | 分化 | 退潮 |
|------|------|------|------|------|------|------|
| 涨停数 | <30 | 30-50 | 50-100 | >100 | 50-100 | <50 |
| 连板高度 | <3 | 3-5 | 5-7 | >7 | 5-3 | <3 |
| 炸板率 | >60% | 40-60% | 30-40% | <30% | 30-50% | >50% |
| 涨跌比 | <0.5 | 0.5-1 | 1-1.5 | >2 | 1-0.5 | <0.5 |
| 成交额 | <1.5T | 1.5-1.8T | >1.8T | >2T | 下降 | <1.5T |

### 3.2 情绪评分表

```
情绪评分 = 涨停数评分 + 连板评分 + 炸板率评分 + 涨跌比评分 + 成交额评分

涨停数评分：
├── >100：25分
├── 50-100：20分
├── 30-50：15分
├── <30：10分

连板评分：
├── >7：25分
├── 5-7：20分
├── 3-5：15分
├── <3：10分

炸板率评分：
├── <30%：25分
├── 30-40%：20分
├── 40-60%：15分
├── >60%：10分

涨跌比评分：
├── >2：25分
├── 1-2：20分
├── 0.5-1：15分
├── <0.5：10分

成交额评分：
├── >2T：25分
├── 1.5-2T：20分
├── 1-1.5T：15分
├── <1T：10分

总分：
├── >100：高潮期
├── 80-100：发酵期
├── 60-80：复苏期
├── 40-60：分化期
├── 20-40：退潮期
├── <20：冰点期
```

## 四、情绪周期实战

### 4.1 周期切换信号

```
冰点 → 复苏
信号：新题材首板增多、昨日涨停溢价、成交量放大

复苏 → 发酵
信号：主线板块出现、龙头股超预期、板块效应明显

发酵 → 高潮
信号：板块持续涨停、成交量创天量、场外资金入场

高潮 → 分化
信号：龙头股炸板、跟风股下跌、炸板率上升

分化 → 退潮
信号：板块集体下跌、跌停股增多、亏钱效应扩散

退潮 → 冰点
信号：连续3天冰点、恐慌性杀跌、成交量萎缩
```

### 4.2 各周期操作策略

```
冰点期：
├── 仓位：0%
├── 策略：空仓等待
├── 只做：观察不做

复苏期：
├── 仓位：20%
├── 策略：轻仓试错
├── 只做：新题材首板
├── 止损：-5%
├── 纪律：不格局

发酵期：
├── 仓位：50%
├── 策略：积极参与
├── 只做：主线板块龙头
├── 加仓：确认后加仓
├── 止损：-5%

高潮期：
├── 仓位：50%
├── 策略：边打边撤
├── 只做：去弱留强
├── 止盈：分批止盈
├── 纪律：不追涨

分化期：
├── 仓位：30%
├── 策略：降低仓位
├── 只做：最强龙头
├── 止盈：锁定利润
├── 纪律：不开新仓

退潮期：
├── 仓位：0-20%
├── 策略：轻仓防守
├── 只做：空仓等待
├── 止损：果断止损
├── 纪律：不抄底
```

### 4.3 周期切换应对

```
周期切换应对：
1. 识别周期切换信号
2. 及时调整仓位
3. 切换操作策略
4. 遵守操作纪律
5. 保持冷静心态

周期切换注意事项：
├── 不要逆周期操作
├── 不要贪念最后一个铜板
├── 不要在退潮期抄底
├── 不要在高潮期追涨
└── 不要在冰点期恐慌
```

```

├── 不选股

├── 等机会

复苏期：
├── 新题材首板
├── 超预期连板

└── 板块唯一涨停

├── 主线板块龙头

├── 换手连板股
└── 放量突破股

├── 板块龙头
├── 补涨龙

├── 最强龙头
```

```
                    │   冰点   │ ──→ 循环
                    └──────────┘
```
