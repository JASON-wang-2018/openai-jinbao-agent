## 一、凯利公式仓位管理

### 公式
```
f = (bp - q) / b
```

### 参数
- **f**：最佳仓位比例
- **b**：赔率（盈亏比）= 预期盈利 / 预期亏损
- **p**：胜率（成功率）
- **q**：败率 = 1 - p

### 示例
- 胜率 50%，盈亏比 2:1
- f = (2×0.5 - 0.5) / 2 = 0.25
- **最佳仓位：25%**

### 简化公式（常用）
```
仓位 = 胜率 - (1 - 胜率) / 盈亏比
```

### 注意事项
- 凯利公式适用于高频交易
- 低频交易建议用半凯利（仓位减半）
- 极端行情需调整参数

## 二、ATR 动态止损

### 计算
```python
ATR = Average(True Range, 14)
止损价 = 入场价 - N × ATR
```

### N 值选择
| 交易风格 | N 值 | 说明 |
|----------|------|------|
| 激进短线 | 1.5 | 止损小，频率高 |
| 稳健短线 | 2.0 | 平衡选择 |
| 中长线 | 2.5 | 止损大，频率低 |

### 动态调整
- **市场波动加大**：ATR 增大 → 扩大止损距离
- **市场波动减小**：ATR 减小 → 收紧止损距离

## 三、波动率仓位管理

### 原理
波动率大的市场 → 低仓位
波动率小的市场 → 高仓位

### 计算
```python
仓位 = 基础仓位 / (ATR / 收盘价 × 100%)
```

### 示例
- 基础仓位 50%
- 股票 A：ATR/收盘 = 2% → 仓位 = 50/2 = 25%
- 股票 B：ATR/收盘 = 5% → 仓位 = 50/5 = 10%

## 五、仓位管理原则

### 1. 总仓位控制
| 市场环境 | 建议仓位 |
|----------|----------|
| 主升趋势 | 80-100% |
| 震荡趋势 | 50-70% |
| 下降趋势 | 0-30% |
| 冰点区域 | 0-10% |

### 2. 单股仓位
| 确定性 | 单股仓位 |
|--------|----------|
| 极高 | 30% |
| 高 | 20% |
| 中 | 10% |
| 低 | 5% |

### 3. 分批建仓
```
第一批：30% （突破买入）
第二批：30% （回踩确认）
第三批：40% （趋势延续）
```

## 七、常见风控误区

1. **不止损** → 一次爆仓归零
2. **频繁止损** → 累积亏损
3. **重仓单一股票** → 黑天鹅风险
4. **逆势加仓** → 亏损加仓
5. **盈利后飘单** → 坐电梯
