## 二、徐翔核心战法

### 2.1 涨停板战法

```
核心理念：
├── 只做涨停板
├── 只做最强的
├── 不做跟风
└── 快进快出

选股标准：
1. 必须是涨停板
2. 必须是板块龙头
3. 必须是市场焦点
4. 成交量必须放大
5. 分时必须强势
```

### 2.2 接力板战法

```
适用场景：
├── 板块持续炒作
├── 龙头连续涨停
├── 市场情绪好

操作要点：
1. 接力必须是换手板
2. 接力必须是板块龙头
3. 接力必须有板块效应
4. 接力必须有市场情绪配合

买入时机：
1. 竞价确认
2. 开盘抢筹
3. 分时放量回封

卖出时机：
1. 收盘不板 → 次日开盘跑
2. 板块退潮 → 竞价跑
3. 低于预期 → 开盘跑
```

### 2.3 低吸战法

```
核心理念：
├── 不是抄底
├── 是低吸强势股回调
├── 是低吸龙头首阴
└── 是低吸分歧低吸

选股标准：
1. 必须是龙头回调
2. 必须是强支撑位
3. 必须是缩量企稳
4. 必须是承接有力

买入时机：
1. 分时急跌到支撑
2. 分时企稳回升
3. 承接有力
4. 量能萎缩

卖出时机：
1. 跌破支撑 → 止损
2. 反弹乏力 → 止盈
3. 达到目标 → 分批跑
```

### 2.4 抢帽子战法

```
核心理念：
├── 开盘抢筹
├── 尾盘出货
├──  T+0操作
└── 日内套利

操作要点：
1. 只做强势股
2. 只做市场焦点
3. 仓位要轻
4. 止损要快

买入条件：
1. 竞价高开 > 3%
2. 开盘快速拉升
3. 承接有力
4. 量能放大

卖出条件：
1. 拉高不板 → 跑
2. 冲高回落 → 跑
3. 收盘不板 → 跑
```

## 四、徐翔仓位管理

### 4.1 仓位原则

```
总仓位控制：
├── 单只个股 ≤ 50%
├── 单个板块 ≤ 70%
├── 总仓位根据行情调整
│   ├── 主升行情：80-100%
│   ├── 震荡行情：50-80%
│   ├── 退潮行情：0-30%
│   └── 冰点行情：0%
└── 新题材试错：≤ 30%
```

### 4.2 加减仓规则

```
首仓：确认后买入30%
确认加仓：超预期后加20%
盈利加仓：浮盈>15%后加20%
分批减仓：盈利>30%减仓1/3
清仓条件：出现见顶信号
```

### 4.3 止损规则

```
固定止损：
├── 首板：-5%
├── 二板：-5%
├── 三板以上：-7%
├── 低吸：-5%
└── 趋势股：-10%

动态止损：
├── 跌破5日线 → 减仓
├── 跌破10日线 → 清仓
├── 收盘不板 → 清仓
├── 板块退潮 → 清仓
└── 出现见顶信号 → 清仓
```

## 五、徐翔复盘习惯

### 5.1 每日复盘

```
复盘内容：
1. 今日涨停复盘
   ├── 涨停原因分析
   ├── 板块效应分析
   ├── 资金流向分析
   └── 次日预期

2. 今日炸板复盘
   ├── 炸板原因分析
   ├── 承接情况分析
   ├── 次日反包预期
   └── 是否值得博弈

3. 今日跌停复盘
   ├── 跌停原因分析
   ├── 是否板块效应
   └── 是否风险扩散

4. 昨日涨停表现
   ├── 收盘涨幅
   ├── 盘中表现
   ├── 次日预期
   └── 是否符合预期
```

### 5.2 每周复盘

```
复盘内容：
1. 本周操作回顾
   ├── 买卖点分析
   ├── 持仓时间分析
   ├── 盈亏情况分析
   └── 心态变化分析

2. 本周市场总结
   ├── 板块轮动总结
   ├── 情绪周期总结
   ├── 赚钱效应总结
   └── 亏钱效应总结

3. 下周计划
   ├── 重点关注板块
   ├── 重点关注个股
   ├── 风险提示
   └── 操作策略
```
