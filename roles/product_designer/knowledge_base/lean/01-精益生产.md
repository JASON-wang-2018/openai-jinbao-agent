## 📖 精益生产概述

### 1. 什么是精益生产？

**Lean Manufacturing** 起源于丰田生产方式 (Toyota Production System, TPS)

**核心理念**:
- 消除一切浪费
- 追求完美
- 持续改进
- 全员参与

**七大浪费 (7 Wastes)**:

| 缩写 | 浪费类型 | 说明 |
|------|----------|------|
| TIMWOOD | Transportation | 运输浪费 |
| I | Inventory | 库存浪费 |
| M | Motion | 动作浪费 |
| W | Waiting | 等待浪费 |
| O | Overproduction | 过度生产 |
| O | Over-processing | 过度加工 |
| D | Defects | 缺陷浪费 |

### 2. VSM绘制步骤

```
1. 选择产品族
2. 绘制当前状态图
3. 计算增值时间比
4. 绘制未来状态图
5. 制定改善计划
```

### 3. 时间计算

**TT (Takt Time)** - 节拍时间:
```
TT = 可用生产时间 / 客户需求量

例如:
- 每天工作8小时 = 480分钟
- 客户需求：240台/天
- TT = 480 / 240 = 2分钟/台
```

**CT (Cycle Time)** - 周期时间:
```
CT = 实际完成一个循环的时间
```

**VA Time** - 增值时间

**NVA Time** - 非增值时间

**增值比**:
```
增值比 = VA Time / (VA Time + NVA Time)
目标: ≥20%
```

## 📦 看板Kanban系统

### 看板类型

| 类型 | 用途 | 特点 |
|------|------|------|
| 生产看板 | 触发生产 | 看板卡/电子看板 |
| 移动看板 | 触发移动 | 物料搬运 |
| 信号看板 | 批量触发 | 触发点/累积点 |
| 领取看板 | 拉动物料 | 下游领取 |

### 看板数量计算

```
看板张数 = (等待时间 × 使用量) / 容器容量 × 安全系数

示例:
- 日需求量：240个
- 补货周期：1天
- 安全系数：1.2
- 容器容量：60个
- 看板张数 = (1 × 240) / 60 × 1.2 = 4.8 ≈ 5张
```

## 📈 产线平衡Line Balancing

### 线平衡计算

**平衡率**:
```
平衡率 = (ΣCT) / (CTmax × N) × 100%

其中:
ΣCT = 各工位时间之和
CTmax = 最大工位时间
N = 工位数

目标: ≥85%
```

**损失时间**:
```
损失时间 = (CTmax × N) - ΣCT
```

### 平衡改善

| 方法 | 说明 | 效果 |
|------|------|------|
| 技能提升 | 减少各工位时间差异 | 降低CTmax |
| 动作分解 | 将瓶颈工位分解 | 减少单工位负荷 |
| 设备辅助 | 设备代替人工 | 标准化CT |
| 并行作业 | 多人协作 | 减少工位数 |

## 🏠 家电行业精益实践

### 案例：洗衣机装配线改善

**背景**:
- 日产能：500台
- 线平衡率：72%
- 换型时间：45分钟

**问题分析**:
| 工位 | CT(秒) | 占比 | 状态 |
|------|--------|------|------|
| 1.上外壳 | 45 | 112% | 瓶颈 |
| 2.装内筒 | 40 | 100% | 满负荷 |
| 3.装电机 | 35 | 87% | 正常 |
| 4.接线 | 30 | 75% | 轻负荷 |
| 5.装面板 | 35 | 87% | 正常 |
| 6.测试 | 40 | 100% | 满负荷 |

**改善措施**:
1. 上外壳工位拆分为两人作业（CT降至30秒）
2. 面板工位增加定位工装（CT降至28秒）
3. 测试工位增加并行测试位（CT降至30秒）
4. 推行SMED，换型时间从45分降至20分

**改善效果**:
| 指标 | 改善前 | 改善后 | 提升 |
|------|--------|--------|------|
| 线平衡率 | 72% | 89% | +17% |
| 日产能 | 500台 | 620台 | +24% |
| 换型时间 | 45分钟 | 20分钟 | -56% |
| 人均效率 | 8.3台/人时 | 10.3台/人时 | +24% |

| 指标 | 计算方法 | 家电行业基准 |

| OEE | 时间开动率×性能开动率×合格品率 | ≥85% |
