## 二、龙头战法核心

### 2.1 什么是龙头

```
龙头定义：
├── 板块中涨幅最大
├── 板块中最早涨停
├── 板块中成交量最大
├── 板块中人气最高
└── 对板块有带动作用

龙头特征：
1. 身位领先（第一个涨停）
2. 盘子适中（20-100亿）
3. 股性活跃（历史涨停多）
4. 题材正宗（受益最直接）
5. 资金合力（多路资金共同参与）
```

### 2.2 龙头战法要点

```
选股标准：
1. 板块效应强（板块>3只涨停）
2. 个股领涨（板块涨幅前3）
3. 成交量放大（量比>1.5）
4. 题材正宗（受益逻辑清晰）
5. 形态良好（均线多头）

买入时机：
1. 竞价抢筹（9:25确认）
2. 开盘秒板（9:30-9:45）
3. 分歧转一致（10:00前）
4. 放量回封（下午2:00后）
5. 题材首板（板块启动首日）

卖出时机：
1. 收盘不板 → 次日竞价跑
2. 板块退潮 → 果断清仓
3. 龙头首阴 → 不格局
4. 低于预期 → 止损
5. 盈利目标 → 分批减仓
```

### 2.3 赵老哥操作模式

```
模式一：板块启动首板
├── 逻辑：板块启动第一天
├── 要点：板块联动、板块指数共振
├── 仓位：20-30%
└── 止损：-5%

模式二：龙头接力
├── 逻辑：板块持续炒作
├── 要点：卡位、换手、人气
├── 仓位：30-50%
└── 止损：-5%

模式三：龙头二波
├── 逻辑：龙头调整后二波
├── 要点：调整缩量、二波启动
├── 仓位：20-30%
└── 止损：-8%

模式四：龙回头
├── 逻辑：龙头首阴后反包
├── 要点：承接有力、大单引导
├── 仓位：20%
└── 止损：-5%
```

## 四、赵老哥仓位管理

### 4.1 分仓原则

```
单只个股：
├── 首板：20-30%
├── 二板：30-40%
├── 三板以上：40-50%
└── 龙头首阴：20%

总仓位：
├── 主升行情：80-100%
├── 震荡行情：50-80%
├── 退潮行情：0-30%
└── 冰点行情：0%
```

### 4.2 加减仓规则

```
首仓：确认后买入20%
确认加仓：次日超预期加20%
盈利加仓：浮盈>20%加10%
分批减仓：盈利>30%减仓1/3
清仓条件：出现见顶信号
```

### 4.3 止损规则

```
固定止损：
├── 首板：-5%
├── 二板：-7%
├── 三板以上：-10%
└── 趋势股：-15%

动态止损：
├── 跌破5日线减仓
├── 跌破10日线清仓
├── 收盘不板止损
└── 板块退潮止损
```

## 五、赵老哥心法

### 5.1 买入逻辑

```
买入前问自己：
1. 这个股票为什么涨？
2. 板块为什么涨？
3. 大盘环境支持吗？
4. 我有优势吗？
5. 错了怎么办？

买入决策：
1. 板块联动 → 买入
2. 板块共振 → 加仓
3. 龙头卡位 → 切换
4. 低于预期 → 止损
5. 超出预期 → 持有
```

### 5.2 卖出逻辑

```
卖出决策：
1. 达到目标 → 减仓
2. 出现风险 → 清仓
3. 板块退潮 → 清仓
4. 低于预期 → 止损
5. 有更好机会 → 切换

不卖出的情况：
1. 涨停板持股
2. 烂板看收盘
3. 板块不退潮
4. 承接有力
5. 浮盈足够
```

### 5.3 复盘习惯

```
每天复盘：
1. 今日涨停复盘
2. 今日炸板复盘
3. 今日跌停复盘
4. 昨日涨停表现
5. 情绪指标统计

每周复盘：
1. 本周操作回顾
2. 本周盈利总结
3. 本周亏损反思
4. 下周机会预判
5. 下周计划制定
```

## 七、实战练习

### 练习题1

```
题目：板块启动第二天，
      板块有5只涨停，
      龙一3连板，
      龙二2连板，
      今日龙二率先涨停，
      如何操作？

答案：
1. 观察龙二是否卡位龙一
2. 卡位成功 → 买入龙二
3. 卡位失败 → 持有龙一
4. 仓位：30%
5. 止损：-5%
```

### 练习题2

```
题目：龙头5连板后，
      今日高开低走放量，
      收出长上影线，
      板块内跟风股大面积炸板，
      如何操作？

答案：
1. 减仓一半
2. 收盘不板清仓
3. 次日不反包彻底清仓
4. 不抄底
5. 等待新周期
```
