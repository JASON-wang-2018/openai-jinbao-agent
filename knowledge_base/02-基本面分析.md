# 基本面分析体系

## 一、财务报表体系

### 1.1 三大报表

| 报表 | 核心内容 | 重点指标 |
|------|---------|---------|
| 资产负债表 | 资产、负债、权益 | 资产负债率、流动比率 |
| 利润表 | 收入、成本、利润 | 营收增速、净利润率 |
| 现金流量表 | 现金流入流出 | 经营现金流、自由现金流 |

### 1.2 资产负债表

#### 资产部分

| 科目 | 含义 | 分析要点 |
|------|------|---------|
| 货币资金 | 现金及存款 | 能否覆盖短期债务 |
| 应收账款 | 客户欠款 | 账期是否合理、回款速度 |
| 存货 | 库存商品 | 周转是否正常 |
| 固定资产 | 厂房设备 | 利用率、更新投入 |
| 无形资产 | 专利、土地 | 价值评估 |

#### 负债部分

| 科目 | 含义 | 分析要点 |
|------|------|---------|
| 短期借款 | 1年内到期 | 偿债压力 |
| 应付账款 | 欠供应商 | 议价能力 |
| 长期借款 | 1年以上 | 财务成本 |
| 应付债券 | 债券融资 | 再融资风险 |

#### 关键指标

```python
# 偿债能力
资产负债率 = 总负债 / 总资产
流动比率 = 流动资产 / 流动负债
速动比率 = (流动资产 - 存货) / 流动负债

# 运营效率
存货周转率 = 营业成本 / 平均存货
应收账款周转率 = 营业收入 / 平均应收账款
总资产周转率 = 营业收入 / 平均总资产
```

### 1.3 利润表

#### 收入端分析

| 指标 | 含义 | 关注点 |
|------|------|--------|
| 营业收入 | 主营业务收入 | 增长趋势 |
| 其他收益 | 补贴、汇兑损益 | 主营聚焦度 |
| 营业外收入 | 非经常收益 | 质量 |

#### 成本费用

| 科目 | 含义 | 分析 |
|------|------|------|
| 营业成本 | 直接成本 | 毛利率 |
| 销售费用 | 营销投入 | 效率 |
| 管理费用 | 运营费用 | 规模效应 |
| 财务费用 | 利息支出 | 债务成本 |
| 研发费用 | 技术投入 | 持续性 |

#### 盈利能力指标

```python
# 毛利率
毛利率 = (营业收入 - 营业成本) / 营业收入

# 净利率
净利率 = 净利润 / 营业收入

# ROE (净资产收益率)
ROE = 净利润 / 股东权益

# ROA (总资产收益率)
ROA = 净利润 / 总资产
```

### 1.4 现金流量表

| 现金流类型 | 来源 | 分析要点 |
|-----------|------|---------|
| 经营活动现金流 | 主营业务的现金流入流出 | 造血能力 |
| 投资活动现金流 | 购建资产、投资收益 | 扩张/收缩 |
| 筹资活动现金流 | 融资、分红 | 资金来源 |

#### 现金流健康度

```python
# 经营现金流/净利润 > 1 → 利润质量高
# 经营现金流/营业收入 > 10% → 回款能力强
# 自由现金流为正 → 造血能力强
自由现金流 = 经营活动现金流 - 资本支出
```

## 二、估值方法

### 2.1 相对估值法

#### PE（市盈率）
```
PE = 股价 / 每股收益(EPS)
PE = 市值 / 净利润
```

| 估值状态 | PE区间 | 含义 |
|---------|--------|------|
| 低估 | <历史中位数 | 可能被低估 |
| 合理 | ≈历史中位数 | 正常估值 |
| 高估 | >历史中位数 | 可能泡沫 |

#### PB（市净率）
```
PB = 股价 / 每股净资产
PB = 市值 / 净资产
```

| 行业 | 低估PB | 合理PB | 高估PB |
|------|--------|--------|--------|
| 银行 | <0.6 | 0.6-1.0 | >1.0 |
| 消费 | <3 | 3-6 | >6 |
| 科技 | <5 | 5-15 | >15 |

#### 股息率
```
股息率 = 股息 / 股价
```

### 2.2 绝对估值法

#### DCF（现金流折现）
```python
# 简化版
内在价值 = Σ(自由现金流 / (1+折现率)^t) + 终值
```

#### 分部估值法
- 不同业务分别估值后加总
- 适用于多元化公司

## 三、行业分析

### 3.1 行业生命周期

```
萌芽期 → 成长期 → 成熟期 → 衰退期
```
- **萌芽期**：技术验证，风险高
- **成长期**：增速最快，投资机会
- **成熟期**：格局稳定，龙头受益
- **衰退期**：需求下降，规避

### 3.2 竞争格局分析

| 格局 | 特征 | 投资策略 |
|------|------|---------|
| 完全竞争 | 分散，无龙头 | 选成本最低 |
| 垄断竞争 | 品牌差异 | 选品牌力强 |
| 寡头垄断 | 少数几家 | 选龙头 |
| 垄断 | 一家独大 | 关注政策 |

### 3.3 波特五力模型

1. **供应商议价能力** - 成本影响
2. **客户议价能力** - 定价能力
3. **新进入者威胁** - 壁垒高低
4. **替代品威胁** - 需求替代
5. **现有竞争者** - 行业激烈度

## 四、宏观经济影响

### 4.1 宏观指标

| 指标 | 影响 | 关注点 |
|------|------|--------|
| GDP增速 | 经济周期 | >6%高增长，<5%增速放缓 |
| CPI | 通胀水平 | >3%通胀，>5%严重通胀 |
| PPI | 生产价格 | 传导到消费端 |
| 利率 | 资金成本 | 利率上升→估值下降 |
| 汇率 | 出口竞争力 | 贬值→出口受益 |

### 4.2 货币政策

| 政策 | 市场影响 | 受益板块 |
|------|---------|---------|
| 降息 | 利好股市 | 金融、地产 |
| 降准 | 释放流动性 | 成长股 |
| 升息 | 利空股市 | 高估值 |
| 量化宽松 | 推升资产价格 | 核心资产 |

### 4.3 财政政策

| 政策 | 市场影响 | 受益板块 |
|------|---------|---------|
| 基建投资 | 刺激经济 | 建材、机械 |
| 减税 | 提升利润 | 消费、科技 |
| 发债 | 流动性增加 | 国债相关 |

## 五、选股框架

### 5.1 自上而下

```
宏观经济 → 行业选择 → 个股筛选
```

1. 判断宏观环境
2. 选择景气行业
3. 挑选行业龙头/弹性标的

### 5.2 自下而上

```
个股研究 → 行业筛选 → 宏观验证
```

1. 发现好公司
2. 判断行业逻辑
3. 验证宏观配合度

### 5.3 好公司的标准

| 维度 | 指标 | 优秀标准 |
|------|------|---------|
| 成长性 | 营收/利润增速 | >15% |
| 盈利性 | ROE | >15% |
| 现金流 | 经营现金流/净利润 | >100% |
| 资产负债 | 资产负债率 | <50% |
| 估值 | PE/PB | 低于行业平均 |
