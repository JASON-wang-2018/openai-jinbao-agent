# 技术分析体系

## 一、K线理论

### 1.1 K线基础
- **阳线**：收盘价>开盘价（红色/绿色看设置）
- **阴线**：收盘价<开盘价
- **上下影线**：日内最高最低价

### 1.2 单根K线意义
| K线类型 | 信号强度 | 含义 |
|---------|---------|------|
| 大阳线 | 强 | 多方主导，可能继续上涨 |
| 大阴线 | 强 | 空方主导，可能继续下跌 |
| 十字星 | 中 | 多空平衡，变盘信号 |
| 锤子线 | 中 | 触底反弹可能 |
| 上吊线 | 中 | 触顶回落可能 |

### 1.3 K线组合
- **红三兵**：三连阳，看涨
- **黑三兵**：三连阴，看跌
- **早晨之星**：底部反转组合
- **黄昏之星**：顶部反转组合
- **乌云盖顶**：顶部反转信号

## 二、趋势分析

### 2.1 趋势类型
- **上升趋势**：高点不断抬高，低点不断抬高
- **下降趋势**：高点不断降低，低点不断降低
- **横盘整理**：无明显趋势

### 2.2 趋势线
- **支撑线**：连接多个低点
- **压力线**：连接多个高点
- **突破**：有效突破需3%或3日确认

### 2.3 趋势阶段
1. **吸筹阶段**：主力建仓，量能温和
2. **拉升阶段**：突破启动，量能放大
3. **出货阶段**：高位震荡，量能萎缩

## 三、形态理论

### 3.1 反转形态

| 形态 | 特征 | 信号 |
|------|------|------|
| 头肩顶 | 左肩、头、右肩 | 强看跌 |
| 头肩底 | 左肩、头、右肩 | 强看涨 |
| 双顶/M顶 | 两个相近高点 | 中看跌 |
| 双底/W底 | 两个相近低点 | 中看涨 |
| 圆弧顶/底 | 渐进转折 | 弱信号 |

### 3.2 持续形态

| 形态 | 特征 | 信号 |
|------|------|------|
| 三角形 | 收敛形态 | 整理后延续 |
| 旗形 | 短边整理 | 快速延续 |
| 楔形 | 收敛后发散 | 变盘信号 |
| 矩形 | 上下震荡 | 整理形态 |

## 四、技术指标

### 4.1 趋势指标

#### MACD（指数平滑异同移动平均线）
```
DIF = EMA(12) - EMA(26)
DEA = EMA(DIF, 9)
MACD = 2 × (DIF - DEA)
```
- **金叉**：DIF上穿DEA → 买入信号
- **死叉**：DIF下穿DEA → 卖出信号
- **背离**：价格创新高/低而MACD不跟随 → 反转信号

#### 布林带（Bollinger Bands）
```
中轨 = MA(20)
上轨 = 中轨 + 2×标准差
下轨 = 中轨 - 2×标准差
```
- **突破上轨**：超买，可能回落
- **跌破下轨**：超卖，可能反弹
- **收口**：波动率降低，可能突破

### 4.2 动量指标

#### RSI（相对强弱指数）
```
RS = n日上涨平均值 / n日下跌平均值
RSI = 100 - (100 / (1 + RS))
```
- **超买**：RSI>70，可能回落
- **超卖**：RSI<30，可能反弹
- **背离**：价格创新高/低而RSI不跟随

#### KDJ随机指标
```
RSV = (收盘价 - n日最低) / (n日最高 - n日最低) × 100
K = 2/3 × 前一日K + 1/3 × RSV
D = 2/3 × 前一日D + 1/3 × K
J = 3K - 2D
```
- **金叉**：K上穿D
- **死叉**：K下穿D
- **超买**：J>100
- **超卖**：J<0

### 4.3 成交量指标

#### 量价关系
- **价涨量增**：健康上涨，趋势延续
- **价涨量缩**：上涨乏力，可能调整
- **价跌量增**：恐慌抛售，可能见底
- **价跌量缩**：阴跌行情，可能持续

#### OBV（能量潮）
- **上升**：资金持续流入
- **下降**：资金持续流出
- **背离**：价格创新高/低而OBV不跟随

## 五、缠论基础（选学）

### 5.1 核心概念
- **走势类型**：上涨、下跌、盘整
- **走势中枢**：重叠价格区间
- **级别**：时间周期的大小

### 5.2 买卖点
- **第一类买卖点**：走势转折点
- **第二类买卖点**：回抽确认点
- **第三类买卖点**：突破中枢点

---

## 六、实战要点

### 6.1 指标配合
- **趋势指标**（MACD、布林带）：判断方向
- **动量指标**（RSI、KDJ）：判断时机
- **成交量**：确认信号

### 6.2 时间周期
- **长周期**（日/周）：判断大趋势
- **中周期**（60分钟）：找入场点
- **短周期**（15分钟/5分钟）：精确买卖

### 6.3 信号过滤
- **多指标共振**：多个指标同时信号更强
- **周期共振**：不同周期同一方向更强
- **量能配合**：有量能支持的信号更可靠
