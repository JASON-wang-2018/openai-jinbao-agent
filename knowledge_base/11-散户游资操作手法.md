## 二、炒股养家心法

### 核心思想

> "买入机会，卖出风险"
> "人气所向，牛股所在"
> "只做龙头，只做主升"

### 情绪周期理论

```
冰点 → 复苏 → 发酵 → 高潮 → 分歧 → 退潮 → 冰点
```

### 养家语录

1. "得散户心者得天下"
2. "买在恐慌，卖在兴奋"
3. "板块高潮时就是卖点"
4. "龙头多命，死在补跌"
5. "高位放量大阴线是危险信号"

### 操作模式

| 模式 | 描述 | 适用场景 |
|------|------|---------|
| 龙头战法 | 买入市场总龙头 | 情绪上升期 |
| 低吸战法 | 恐慌低吸核心标的 | 情绪冰点 |
| 接力战法 | 买入换手龙头 | 情绪发酵期 |
| 反包战法 | 买入反包板 | 分歧转一致 |

## 五、北京炒家

### 核心思想

> "首板是龙头的起点"
> "每天都在寻找下一个龙头"

### 首板战法

**首板选股标准**：

```
1. 前一交易日涨停（非一字）
2. 竞价高开3-7%
3. 成交量放大
4. 板块效应
```

**首板买入时机**：

- 竞价买入（强势票）
- 开盘快速拉升时买入
- 炸板回封买入

## 七、KK（竞价战法）

### 核心思想

> "竞价定方向"
> "开盘十五分钟定全天"

### 竞价战法

**竞价看盘要点**：

| 信号 | 含义 |
|------|------|
| 竞价抢筹 | 高开3%+，抢筹明显 |
| 竞价抢跑 | 高开低走，抢跑 |
| 竞价核按钮 | 跌停竞价 |
| 竞价放量 | 成交放量 |

**竞价操作**：

```
强 → 竞价买入
弱 → 竞价卖出
```

## 十、游资战法总结

### 共同点

1. **只做最强** - 只做龙头
2. **情绪理解** - 理解市场情绪
3. **严格止损** - 错了就卖
4. **仓位管理** - 控制回撤
5. **复盘习惯** - 每天复盘

### 不同点

| 维度 | 超短派 | 中线派 |
|------|--------|--------|
| 持股周期 | 1-5天 | 1-12周 |
| 交易频率 | 高 | 低 |
| 选股 | 技术面+情绪 | 基本面+趋势 |
| 止损 | 3-5% | 10-15% |
| 止盈 | 5-15% | 30%+ |

## 十一、散户vs游资

### 散户常见错误

| 错误 | 表现 |
|------|------|
| 追涨杀跌 | 高买低卖 |
| 频繁交易 | 过度交易 |
| 死扛不止损 | 亏损死拿 |
| 盈利就跑 | 小赚大亏 |

### 游资优势

| 优势 | 说明 |
|------|------|
| 理解情绪 | 把握市场情绪周期 |
| 纪律执行 | 严格止损 |
| 仓位控制 | 控制回撤 |
| 复盘习惯 | 每天总结 |

## 十二、实战技巧

### 游资票识别

```
特征1：涨停板吸筹
特征2：连板上涨
特征3：高位震荡出货
特征4：龙虎榜机构席位
```

### 跟游资操作

```
1. 关注龙虎榜数据
2. 分析席位操作风格
3. 跟买要趁早
4. 跌破止损位要跑
```

### 规避游资票

```
1. 高位放量滞涨
2. 频繁上榜但涨幅有限
3. 出现多个一线游资对倒
4. 板块退潮期抢跑
```
