# 超级短线战法

> 超级短线，又称"超短"，持股周期1-3天，甚至日内。
> 核心逻辑：复利增长。

---

## 一、超短核心原理

### 1.1 超短的本质

```
超短 = 情绪博弈 + 概率优势 + 复利增长

超短公式：
账户资金 = 初始资金 × (1 + 每次盈利)ⁿ × (1 - 每次亏损)ᵐ

核心要点：
├── 小资金做大 → 必须超短
├── 复利增长 → 稳定盈利
├── 概率优势 → 胜率 × 盈亏比
└── 情绪博弈 → 把握情绪周期
```

### 1.2 超短四要素

| 要素 | 内容 | 重要性 |
|------|------|--------|
| **情绪** | 把握情绪周期 | ⭐⭐⭐⭐⭐ |
| **板块** | 板块联动效应 | ⭐⭐⭐⭐ |
| **个股** | 龙头、连板、人气 | ⭐⭐⭐⭐ |
| **节奏** | 买入、卖出时机 | ⭐⭐⭐⭐⭐ |

## 二、情绪周期理论

### 2.1 六阶段模型

```
冰点 → 复苏 → 发酵 → 高潮 → 分化 → 退潮 → 冰点
  ↓      ↓      ↓      ↓      ↓      ↓
  1-3天  2-4天  2-4天  1-3天  1-2天  1-4天
```

### 2.2 各阶段特征

| 阶段 | 涨停数 | 炸板率 | 情绪 | 操作 |
|------|--------|--------|------|------|
| 冰点 | <30 | >60% | 恐慌 | 空仓 |
| 复苏 | 30-50 | 40-60% | 犹豫 | 轻仓试错 |
| 发酵 | 50-100 | 30-40% | 转强 | 积极参与 |
| 高潮 | >100 | <30% | 亢奋 | 边打边撤 |
| 分化 | 50-100 | 30-50% | 分歧 | 去弱留强 |
| 退潮 | <50 | >50% | 转弱 | 降低仓位 |

## 三、竞价战法

### 3.1 竞价选股

```
竞价看什么：
1. 昨日涨停表现
2. 今日竞价涨幅
3. 竞价量能
4. 板块竞价
5. 龙头竞价

竞价买入条件：
├── 竞价涨幅 > 3%
├── 板块竞价涨幅 > 1%
├── 板块内多只高开
├── 个股承接有力
└── 大盘环境配合

仓位：20-30%

止损：-3%
```

### 3.2 竞价卖股

```
竞价卖出条件：
1. 竞价涨幅 < -3%
2. 板块整体弱势
3. 个股承接无力
4. 大盘环境恶劣
5. 达到目标位

操作：
├── 符合条件 → 竞价跑
├── 不及预期 → 开盘跑
└── 超预期 → 持有
```

```
3. 9:35-10:00 第一小时

├── 板块联动
├── 量能放大
├── 承接有力
└── 大盘配合

仓位：30-40%

止损：-3%
```

### 4.2 开盘卖股

```
开盘卖出条件：
1. 开盘不及预期
2. 承接无力
3. 快速下跌
4. 板块走弱
5. 大盘跳水

操作：
├── 开盘3分钟内卖出
├── 不要幻想反弹
├── 不要格局
└── 止损要快
```

## 五、盘中战法

### 5.1 分时看盘

```
分时关键点：
1. 均价线（黄线）→ 成本线
2. 量峰 → 资金强弱
3. 承接 → 买盘力量
4. 抛压 → 卖盘压力
5. 情绪 → 市场氛围

分时形态：
├── 放量突破 → 买入
├── 回踩不破 → 买入
├── 承接有力 → 买入
├── 放量滞涨 → 卖出
├── 破位下跌 → 卖出
└── 抛压过大 → 卖出
```

### 5.2 盘中选股

```
盘中买入条件：
1. 板块联动（多只涨停）
2. 个股领涨（板块涨幅前3）
3. 分时放量（量比>2）
4. 承接有力（均价线上）
5. 大盘配合（指数翻红）

仓位：30-40%

止损：-3%
```

### 5.3 盘中卖股

```
盘中卖出条件：
1. 涨停打开（2次以上）
2. 放量滞涨（量能放大，不涨）
3. 跌破均价线
4. 板块走弱
5. 大盘跳水

操作：
├── 炸板2次 → 卖出1/2
├── 跌破均价 → 卖出1/2
├── 收盘不板 → 清仓
└── 板块退潮 → 清仓
```

## 七、连板战法

### 7.1 首板战法

```
首板条件：
1. 板块启动（>2只涨停）
2. 个股领涨（板块涨幅前3）
3. 题材正宗（逻辑硬）
4. 形态良好（均线多头）
5. 量能健康（温和放量）

买入时机：
1. 竞价确认
2. 开盘秒板
3. 分时回封

仓位：20-30%

止损：-5%

卖出时机：
1. 次日不板 → 开盘跑
2. 低于预期 → 开盘跑
3. 达到目标 → 分批跑
```

### 7.2 二板战法

```
二板条件：
1. 首板质量（换手板）
2. 板块持续（>2只连板）
3. 市场情绪回暖
4. 承接有力
5. 分时稳定

买入时机：
1. 竞价超预期
2. 开盘快速涨停
3. 分时放量回封

仓位：30-40%

止损：-5%

卖出时机：
1. 收盘不板 → 次日开盘跑
2. 低于预期 → 次日开盘跑
3. 板块退潮 → 次日开盘跑
```

### 7.3 三板及以上的战法

```
三板条件：
1. 市场空间板
2. 板块绝对龙头
3. 市场情绪配合
4. 题材级别大
5. 人气旺盛

买入时机：
1. 竞价确认
2. 开盘抢筹
3. 放量回封

仓位：40-50%

止损：-7%

卖出时机：
1. 收盘不板 → 次日竞价跑
2. 炸板2次 → 次日竞价跑
3. 板块退潮 → 次日竞价跑
4. 出现见顶信号 → 坚决跑
```

## 八、情绪指标

### 8.1 情绪指标表

| 指标 | 乐观 | 中性 | 悲观 |
|------|------|------|------|
| 涨停数 | >100 | 50-100 | <50 |
| 连板高度 | >7 | 3-7 | <3 |
| 炸板率 | <30% | 30-50% | >50% |
| 涨跌比 | >2 | 1-2 | <1 |
| 成交额 | >1万亿 | 7000-10000亿 | <7000亿 |
| 北向资金 | >50亿 | -50~50亿 | <-50亿 |

### 8.2 情绪判断方法

```
每日情绪判断：
1. 昨日涨停表现
2. 今日竞价表现
3. 今日盘中表现
4. 今日收盘表现
5. 明日预期

情绪评分：
0-20分：冰点 → 空仓
20-40分：偏冰 → 轻仓
40-60分：中性 → 半仓
60-80分：偏热 → 积极参与
80-100分：过热 → 边打边撤
```

## 十、止损止盈

### 10.1 止损规则

```
固定止损：
├── 首板：-5%
├── 二板：-5%
├── 三板以上：-7%
├── 低吸：-5%
└── 趋势：-10%

动态止损：
├── 跌破5日线 → 减仓
├── 跌破10日线 → 清仓
├── 收盘不板 → 清仓
├── 板块退潮 → 清仓
└── 出现见顶信号 → 清仓
```

### 10.2 止盈规则

```
分批止盈：
├── 盈利10% → 减仓1/3
├── 盈利20% → 减仓1/3
├── 盈利30% → 减仓1/3
├── 盈利50% → 再减仓1/2
└── 剩余仓位 → 格局

目标止盈：
├── 首板：次日10%
├── 二板：次日15%
├── 三板：次日20%
├── 四板以上：不设目标
└── 低吸：次日8%
```

## 十一、实战案例

### 案例1：冰点转折首板

```
背景：市场连续冰点3天
标的：太平洋

操作：
1. 竞价发现板块异动
2. 板块3只涨停
3. 竞价涨幅8%
4. 开盘介入
5. 次日高开持有
6. 3板后清仓

盈利：>80%

核心：
- 把握冰点转折
- 板块联动
- 竞价确认
```

### 案例2：龙头接力

```
背景：板块持续炒作
标的：剑桥科技

操作：
1. 首板介入
2. 二板加仓
3. 三板持有
4. 四板减仓
5. 五板清仓

盈利：>200%

核心：
- 龙头信仰
- 持续加仓
- 高位减仓
```

## 十二、常见错误

| 错误 | 正确做法 |
|------|---------|
| 竞价不及预期还拿着 | 不及预期 → 开盘跑 |
| 开盘弱势不止损 | 开盘3分钟跑 |
| 炸板后格局 | 炸板2次跑 |
| 收盘不板还拿着 | 收盘不板 → 明天跑 |
| 板块退潮不止损 | 板块退潮 → 清仓 |
| 盈利后急于卖出 | 达到目标再卖 |
| 亏损后急于翻本 | 冷静后再操作 |
| 逆势操作 | 只做主流 |
