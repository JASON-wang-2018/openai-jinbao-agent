## 一、缠论基础概念

### 1.1 核心思想

```
缠论核心思想：
├── 走势必完美
├── 中枢定义方向
├── 背驰判断终结
├── 级别决定操作
└── 三类买卖点定位
```

### 1.2 走势类型

```
上涨走势：中枢不断上移
下跌走势：中枢不断下移
盘整走势：中枢不创新高/新低
趋势走势：中枢不断创新高/新低
```

### 1.3 级别概念

| 级别 | 时间周期 | 操作意义 |
|------|----------|----------|
| 1分钟 | 1-5分钟 | 超短线 |
| 5分钟 | 1-4小时 | 短线 |
| 30分钟 | 1-3天 | 波段 |
| 日线 | 1-4周 | 中线 |
| 周线 | 1-3月 | 长线 |
| 月线 | 3-12月 | 长线 |

## 二、中枢理论

### 2.1 中枢定义

```
中枢：由连续三段重叠的走势类型组成

特征：
├── 至少三段连续的次级别走势
├── 后一段的高点 > 前一段的低点
├── 后一段的低点 < 前一段的高点
└── 中枢区间：高点中的低点 ~ 低点中的高点

示意图：
        ┌───┐
    ┌───┘   └───┐
    │           │
┌───┘           └───┐
│                   │
└───────────────────┘
    连续三段重叠
```

### 2.2 中枢级别

| 级别 | 幅度 | 持续时间 | 意义 |
|------|------|----------|------|
| 笔中枢 | 3-5% | 3-5天 | 小级别 |
| 段中枢 | 5-15% | 1-3周 | 中级别 |
| 笔数中枢 | 15-30% | 1-3月 | 大级别 |
| 形态中枢 | >30% | 3-12月 | 超级中枢 |

### 2.3 中枢方向

```
中枢方向判定：
├── 中枢区间 > 前一高点 → 上涨中枢
└── 中枢区间 < 前一低点 → 下跌中枢
```

### 2.4 中枢操作

```
中枢震荡：高抛低吸
中枢突破：顺势操作
中枢延伸：继续震荡
中枢扩张：升级级别
```

## 三、走势类型

### 3.1 上涨走势

```
特征：
├── 低点不断抬高
├── 高点不断创新
└── 围绕中枢震荡向上

操作策略：
├── 回调中枢上沿买入
├── 突破中枢回踩买入
└── 不破中枢上沿持有
```

### 3.2 下跌走势

```
特征：
├── 高点不断降低
├── 低点不断创新
└── 围绕中枢震荡向下

操作策略：
├── 反弹中枢下沿卖出
├── 跌破中枢反抽卖出
└── 不破中枢下沿观望
```

### 3.3 盘整走势

```
特征：
├── 围绕中枢震荡
├── 不创新高/新低
└── 区间整理

操作策略：
├── 中枢上沿卖出
├── 中枢下沿买入
├── 突破中枢跟随
└── 跌破中枢止损
```

## 四、背驰理论

### 4.1 背驰定义

```
背驰：走势力度减弱，导致走势类型终结

分类：
├── 盘整背驰：创新高/新低后背驰
├── 趋势背驰：趋势走势中的背驰
└── 小级别背驰：相邻同级别走势力度对比
```

### 4.2 背驰判断

```
判断方法：
1. 力度对比（MACD面积）
2. 幅度对比
3. 时间对比
4. 成交量对比

背驰条件：
├── 创新高/新低
├── 力度明显减弱
├── 斜率变缓
└── 成交量萎缩
```

### 4.3 背驰级别

| 背驰级别 | 力度减弱 | 转折概率 | 操作意义 |
|----------|----------|----------|----------|
| 小级别 | <30% | 50% | 小反弹 |
| 中级别 | 30-50% | 70% | 较大反弹 |
| 大级别 | >50% | >90% | 趋势反转 |

## 五、三类买卖点

### 5.1 第一类买点

```
定义：走势类型终结前的最低点

特征：
├── 背驰后产生
├── 下跌走势的终点
├── 买入的最佳时机

确认方法：
├── 背驰确认
├── 次级别走势不创新低
└── 出现底分型
```

### 5.2 第二类买点

```
定义：第一类买点后的第一次回调不破低点

特征：
├── 第一类买点后的回抽
├── 不破第一类买点
├── 买入的确认时机

优势：
├── 相比一买更低的风险
├── 更明确的买入信号
└── 更高的确定性
```

### 5.3 第三类买点

```
定义：中枢突破后的回抽不破中枢上沿

特征：
├── 中枢突破后的回抽
├── 不回到中枢区间
├── 趋势延续的确认

操作意义：
├── 确认趋势延续
├── 趋势交易买入点
├── 追涨买入点
```

### 5.4 三类卖点

```
与三类买点相反：

第一类卖点：上涨走势的终点
第二类卖点：第一类卖点后的第一次反抽不破高点
第三类卖点：中枢跌破后的反抽不回到中枢区间
```

### 5.5 买卖点图示

```
买入点位置：

第一类买点     第二类买点     第三类买点
    ↓            ↓            ↓
    ┌─┐         ┌─┐         ┌─┐
    │ │         │ │         │ │
────┘ └─────────┘ └─────────┘ └─────
  背驰点    回抽不破      突破回抽

卖点位置相反。
```

## 六、笔与线段

### 6.1 笔的定义

```
笔：相邻两个顶底分型之间的走势

笔的构成：
├── 顶分型 + 中间K线 + 底分型
├── 顶分型的高点 > 底分型的低点
└── 笔的力度：顶 - 底

笔的合并：
├── 属于同一笔的K线
├── 相邻两个顶分型合并为更高的顶
└── 相邻两个底分型合并为更低的底
```

### 6.2 线段的定义

```
线段：三笔重叠构成

特征：
├── 至少三笔连续重叠
├── 线段有明确的涨跌幅
└── 线段代表次级别走势

线段终结：
├── 被相反线段破坏
├── 出现笔破坏
└── 中枢扩张
```

### 6.3 笔与线段的关系

```
笔 → 线段 → 中枢 → 走势类型

层级：
├── 笔：最小单位
├── 线段：由笔构成
├── 中枢：由线段构成
└── 走势类型：由中枢构成
```

## 七、走势完美

### 7.1 完美定义

```
走势必完美：
├── 任何走势终将完成
├── 完成后必然转折
└── 转折后必然形成新走势

意义：
├── 走势有始有终
├── 转折可预测
└── 操作有依据
```

### 7.2 完美应用

```
上涨走势完美：
├── 必然形成中枢
├── 必然创新高
└── 必然出现背驰

下跌走势完美：
├── 必然形成中枢
├── 必然创新低
└── 必然出现背驰
```

## 八、级别操作

### 8.1 级别选择

```
操作级别选择：
├── 小级别（1-5分钟）：超短线
├── 中级别（30分钟-日线）：短线/波段
├── 大级别（周线）：中线/长线
└── 级别匹配：根据资金规模选择
```

### 8.2 级别转换

```
不同级别的关系：
├── 5个1分钟走势 = 1个5分钟走势
├── 3个5分钟走势 = 1个15分钟走势
├── 4个15分钟走势 = 1个60分钟走势
└── 4个60分钟走势 = 1个日线走势
```

### 8.3 多级别联立

```
联立分析：
├── 小级别：精确买卖点
├── 中级别：趋势方向
└── 大级别：仓位控制

操作原则：
├── 大级别向上 → 中级别回调买入
├── 中级别向上 → 小级别回抽买入
└── 小级别出现背驰 → 减仓/清仓
```

## 九、实战应用

### 9.1 选股标准

```
基本面：
├── 行业处于上升周期
├── 公司业绩增长
└── 无重大利空

技术面：
├── 日线级别中枢突破
├── 周线级别盘整背驰
└── 月线级别底分型确认
```

### 9.2 买入策略

```
第一类买点买入：
├── 等待日线级别下跌走势完美
├── 30分钟级别背驰确认
└── 成交量明显萎缩

第二类买点买入：
├── 第一类买点后的回抽不破
├── 30分钟级别底分型确认
└── 成交量温和放大

第三类买点买入：
├── 中枢突破后的回抽
├── 不回到中枢区间
└── 成交量放大配合
```

### 9.3 卖出策略

```
第一类卖点卖出：
├── 日线级别上涨走势完美
├── 30分钟级别背驰确认
└── 成交量明显放大

第二类卖点卖出：
├── 第一类卖点后的反抽不破
├── 30分钟级别顶分型确认
└── 成交量萎缩

第三类卖点卖出：
├── 中枢跌破后的反抽
├── 不回到中枢区间
└── 成交量萎缩
```

## 十一、缠论精髓

### 11.1 核心口诀

```

三类定买卖，级别决仓位。
走势必完美，买卖有依据。
```

### 11.2 实战口诀

```
下跌背驰找一买，反弹不破找二买。
中枢突破找三买，趋势延续有保证。
上涨背驰找一卖，反抽不破找二卖。
中枢跌破找三卖，趋势终结要跑路。
```

### 11.3 操作原则

```
1. 只做确定性的买卖点
2. 只操作与操作级别同向的走势
3. 只操作本级别中枢震荡的股票
4. 背驰后的转折必须参与
5. 走势完美后必须退出
```
