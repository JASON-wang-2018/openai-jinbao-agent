## 一、战略管理

### 1.1 战略分析工具

**SWOT分析：**
| 维度 | 内容 |
|------|------|
| S (优势) | 核心竞争力、资源优势 |
| W (劣势) | 短板、待改进 |
| O (机会) | 市场趋势、政策利好 |
| T (威胁) | 竞争对手、替代品 |

**波特五力模型：**
1. 行业内现有竞争者
2. 潜在进入者
3. 替代品威胁
4. 供应商议价能力
5. 买方议价能力

### 1.2 战略执行

**平衡计分卡 (BSC)：**
| 维度 | 指标 |
|------|------|
| 财务 | 营收、利润、增长率 |
| 客户 | 满意度、市场份额 |
| 内部流程 | 效率、质量 |
| 学习成长 | 员工技能、创新能力 |

## 二、组织行为学

### 2.1 组织结构类型

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 职能型 | 按职能划分部门 | 稳定环境 |
| 事业部 | 按产品/区域划分 | 多元化业务 |
| 矩阵型 | 双重汇报 | 项目制 |
| 扁平化 | 层级少 | 创新型 |

### 2.2 组织文化

**三种类型：**
1. **强人文化** - 竞争、创新（投行、咨询）
2. **努力工作/尽情享受** - 销售、服务
3. **流程文化** - 稳定、政府机构

## 三、绩效管理

### 3.1 KPI vs OKR

| 维度 | KPI | OKR |
|------|-----|-----|
| 目标 | 达成 | 挑战 |
| 数量 | 少而精 | 多而广 |
| 评分 | 100%完成 | 60-70%算成功 |
| 周期 | 月/季 | 季度 |

### 3.2 绩效评估方法

- **360度评估**：上级+同级+下级+客户
- **MBO**：目标管理
- **强制分布**：末位淘汰
- **校准会议**：确保公平

---

## 四、财务管理

### 4.1 核心财务指标

| 指标 | 公式 | 含义 |
|------|------|------|
| 毛利率 | (收入-成本)/收入 | 盈利能力 |
| 净利率 | 净利润/收入 | 最终盈利 |
| ROE | 净利润/股东权益 | 投资回报 |
| 资产负债率 | 负债/资产 | 财务风险 |

### 4.2 预算管理

**零基预算 (ZBB)：**
- 不基于历史数据
- 每个项目需证明必要性
- 适用于职能部门

## 五、变革管理

### 5.1 变革阻力来源

1. 利益冲突
2. 习惯惯性
3. 认知偏差
4. 资源不足

### 5.2 变革推动模型

**Kotter八步法：**
1. 制造紧迫感
2. 建立联盟
3. 形成愿景
4. 广泛沟通
5. 授权行动
6. 创造短期成果
7. 巩固成果
8. 融入文化

## 六、领导力发展

### 6.1 领导力模型

**情境领导力：**
| 下属成熟度 | 领导风格 |
|------------|----------|
| D1 (低能力/高意愿) | S1 指挥型 |
| D2 (低能力/低意愿) | S2 教练型 |
| D3 (高能力/低意愿) | S3 支持型 |
| D4 (高能力/高意愿) | S4 授权型 |

### 6.2 教练式领导

**GROW模型扩展：**
- **G**oal - 目标
- **R**eality - 现状
- **O**ptions - 选项
- **W**ill - 意愿 + **W**ay forward - 行动计划

## 七、谈判技巧

### 7.1 谈判策略

| 策略 | 场景 | 技巧 |
|------|------|------|
| 硬式 | 立场对立 | 施压、最后通牒 |
| 软式 | 关系重要 | 让步、求同 |
| 原则 | 双赢 | 利益vs立场 |

### 7.2 谈判技巧

- **锚定效应**：先报价
- **让步策略**：递减式
- **最后通牒**：设定deadline
- **黑脸白脸**：两人配合

## 八、危机管理

### 8.1 危机处理原则

1. **快速响应** - 第一时间回应
2. **统一口径** - 一个出口
3. **事实为基** - 不说谎
4. **承担责任** - 不推卸

### 8.2 沟通策略

| 对象 | 策略 |
|------|------|
| 内部 | 稳定军心、统一行动 |
| 媒体 | 主动发声、掌控话语权 |
| 公众 | 真诚道歉+解决方案 |
| 监管 | 积极配合、主动报告 |
