## 一、核心投资理念

### 1.1 曹明成的投资信条

> "在中国的资本市场，我从来不讲价值投资。所谓的价值，不过是给庄家炒作的理由而已。我选股思路是跟庄，操作理论讲究趋势为先。"

- **核心理念**：跟庄 + 趋势为先
- **买入点**：次低点或次次低点
- **卖出点**：次高点或次次高点

### 1.2 曹氏八线理论

**核心口诀**：**线上持股，线下持币**

- 以10日线为基准
- 趋势形成后，形态明朗后才操作
- 不等同于右侧交易（追求更优的买入价位）
- 下降通道必须空仓观望

## 二、庄家运作流程

```
建仓 → 洗盘 → 拉升 → 出货
```

### 2.1 建仓阶段（获取筹码）

**三种主要方式**：
1. **砸盘吸筹** - 连续下跌制造恐慌
2. **拉高抢筹** - 快速拉升收集筹码
3. **横盘震荡吸筹** - 长时间横盘整理

**盘面特征**：
- 日K线底部放量
- 收盘经常收下影线
- 均线粘合后向上发散

### 2.2 洗盘阶段

**目的**：
- 清洗浮筹
- 提高市场成本
- 为拉升做准备

**手法**：
- **"磨"字诀** - 长时间横盘震荡
- **"打"字诀** - 快速下跌制造恐慌

### 2.3 拉升阶段

**时机选择**：
- 大盘配合
- 题材成熟
- 筹码稳定

**拉升方式**：
- **慢牛式** - 稳步推高
- **快牛式** - 连续涨停

### 2.4 出货阶段

**特征**：
- 放量滞涨
- 高位长上影
- 跌破重要均线

## 三、主力行为识别

### 3.1 底部放量信号

- 底部突然放量
- 持续性资金流入
- 同类题材集体异动

### 3.2 题材炒作识别

**常见题材**：
- 政策利好
- 业绩预增
- 资产重组
- 新技术/新概念

**应对策略**：
- 不追已经大涨的题材
- 寻找低位刚启动的标的

### 3.3 盘口异动

- 收盘前快速拉升
- 整数关口大单护盘
- 主动卖单减少

## 五、财报陷阱识别

### 5.1 常见造假手法

| 方法 | 手段 |
|------|------|
| 虚增收入 | 伪造销售合同 |
| 提前确认收入 | 提前计入营收 |
| 转移费用 | 费用资本化 |
| 虚增资产 | 资产评估增值 |

### 5.2 识别技巧

1. **毛利分析法** - 异常高毛利需警惕
2. **现金流量法** - 经营现金流与利润不匹配
3. **应收款项法** - 应收账款异常增长
4. **存货分析法** - 存货周转异常

## 六、实战战法

### 6.1 天梯抓主升

**定义**：设置特殊均线组合捕捉主升浪

**核心**：
- 识别庄家拉升意图
- 持股待涨
- 跌破均线退出

### 6.2 长牛线抓长线

**原理**：
- 庄家最怕长期持股不卖
- 长牛线是庄家成本线

## 七、与双系统模型整合

### 7.1 理念对应

| 曹氏理论 | 双系统模型 |
|----------|------------|
| 趋势为先 | 主升系统（指数强趋势） |
| 跟庄操作 | 寻找主线板块龙头 |
| 10日线 | 辅助判断短期趋势 |

### 7.2 实战结合

1. **主升系统筛选** → 确认大盘和板块趋势
2. **跟庄思维选股** → 寻找主力资金流入的个股
3. **10日线辅助** → 把握短线买卖点
4. **出货信号识别** → 避免追高被套

## 八、核心要点总结

1. **认识主力**：了解庄家运作规律
2. **趋势为王**：线上持股，线下持币
3. **跟庄思维**：与主力同步，而非对抗
4. **风险意识**：识别出货信号，及时止盈止损
5. **信息辨别**：不被庄家信息战误导
