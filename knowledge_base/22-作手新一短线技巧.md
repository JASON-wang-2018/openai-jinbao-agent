## 二、作手新一核心模式

### 2.1 首板模式

```
模式特点：
├── 风险较低
├── 确定性较高
├── 收益适中
└── 适合大多数投资者

选股标准：
1. 板块联动（板块>2只涨停）
2. 个股领涨（板块涨幅前3）
3. 题材正宗（逻辑硬）
4. 形态良好（均线多头）
5. 量能健康（温和放量）

买入时机：
1. 竞价确认（9:25）
2. 开盘秒板（9:30-9:35）
3. 分时回封（10:00前）
4. 午后转势板（14:00后）

卖出时机：
1. 次日不板 → 开盘跑
2. 低于预期 → 开盘跑
3. 板块退潮 → 收盘跑
4. 达到目标 → 收盘跑

仓位：20-30%

止损：-5%
```

### 2.2 接力模式

```
模式特点：
├── 确定性高
├── 收益较高
├── 风险较高
└── 需要大盘配合

选股标准：
1. 昨日涨停（非一字）
2. 换手充分（换手率5-15%）
3. 板块效应（板块>2只连板）
4. 情绪回暖（炸板率<40%）
5. 形态突破（突破整理平台）

买入时机：
1. 竞价高开（>3%）
2. 开盘快速上板（9:30-9:45）
3. 分时放量分歧转一致
4. 午后回封

仓位：30-40%

止损：-5%
```

### 2.3 龙头模式

```
模式特点：
├── 确定性最高
├── 收益最高
├── 风险最高
└── 适合高手

选股标准：
1. 市场总龙头（最高连板）
2. 板块绝对龙头
3. 成交额>10亿
4. 市场情绪配合
5. 题材级别大

买入时机：
1. 竞价确认（超预期）
2. 开盘抢筹（承接强）
3. 分时低吸（回调到均线）
4. 放量回封（确定性高）

仓位：40-50%

止损：-7%
```

### 2.4 低吸模式

```
模式特点：
├── 风险较低
├── 收益适中
└── 确定性一般

选股标准：
1. 龙头回调（首阴/双阴）
2. 支撑位（MA5/MA10/平台）
3. 量能萎缩（回调缩量）
4. 承接有力（分时不破位）
5. 逻辑不变（题材继续看好）

买入时机：
1. 分时急跌到支撑
2. 分时企稳回升
3. 量能萎缩
4. 承接有力

仓位：20-30%

止损：-5%
```

## 三、情绪周期应用

### 3.1 冰点期

```
特征：
├── 涨停家数<30
├── 连板高度<3
├── 炸板率>50%
├── 跌停家数>10
└── 成交额萎缩

操作策略：
1. 空仓等待
2. 轻仓试错（20%）
3. 只做最强的首板
4. 快进快出
5. 不格局

关注信号：
1. 首板数量增加
2. 连板高度突破
3. 炸板率下降
4. 出现板块效应
```

### 3.2 复苏期

```
特征：
├── 涨停家数30-50
├── 连板高度3-5
├── 炸板率30-50%
├── 成交额放大
└── 出现主线板块

操作策略：
1. 逐步加仓（50%）
2. 参与主线板块
3. 参与板块龙头
4. 适当格局
5. 止损果断

关注信号：
1. 板块持续性
2. 龙头高度
3. 跟风股表现
```

### 3.3 发酵期

```
特征：
├── 涨停家数50-100
├── 连板高度5-7
├── 炸板率<30%
├── 成交额放大
├── 主线板块明确
└── 市场情绪亢奋

操作策略：
1. 重仓参与（80%）
2. 只做主线
3. 只做龙头
4. 让利润奔跑
5. 边打边撤

风险控制：
1. 关注高位风险
2. 关注退潮信号
3. 分批止盈
4. 不追高位
```

### 3.4 高潮期

```
特征：
├── 涨停家数>100
├── 连板高度>7
├── 炸板率<20%
├── 成交额天量
├── 全民亢奋
└── 各种补涨

操作策略：
1. 边打边撤
2. 去弱留强
3. 锁定利润
4. 不开新仓
5. 准备空仓

退潮信号：
1. 龙头炸板率高
2. 跟风股大面积炸板
3. 跌停股增加
4. 成交额萎缩
```

### 3.5 退潮期

```
特征：
├── 涨停家数下降
├── 连板高度下降
├── 炸板率上升
├── 跌停股增加
├── 成交额萎缩
└── 亏钱效应明显

操作策略：
1. 降低仓位（30%）
2. 只做首板
3. 快进快出
4. 止损果断
5. 准备空仓

关注信号：
1. 何时见底
2. 新题材出现
3. 冰点信号
```
