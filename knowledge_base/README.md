## 四、核心框架速查

### 6步技术分析工作流（08文件）

```
1. 大盘趋势判断 → 当前处于什么阶段？
2. 板块强度分析 → 哪个板块最强？
3. 个股技术信号 → 出现什么信号？
4. 量价关系验证 → 成交量是否配合？
5. 买卖点确认 → 在哪里买入/卖出？
6. 风险评估 → 仓位和止损怎么设？
```

### 双系统模型（25文件）

```
主升系统 v3.0（强趋势行情）
├─ 指数强趋势：MA20>MA60+指数>MA10
├─ 主线板块：强于指数1.05倍
├─ 龙头优先：近5日新高+涨停
├─ 量价强一致：放量突破+回调缩量
├─ 分歧确认：第一天分歧→第二天转强
└─ 失败压制：无高位加速/巨量阴

冰点系统 v1.0（弱势反转行情）
├─ 情绪冰点四要素
├─ 逆势首板先锋
└─ 轻仓试错
```

### 庄家坐庄流程（09/34文件）

```
选股 → 建仓 → 洗盘 → 试盘 → 拉升 → 出货 → 砸盘
```

### 情绪周期（20/27文件）

```
冰点期 → 复苏期 → 发酵期 → 高潮期 → 分化期 → 退潮期 → 冰点期
```
