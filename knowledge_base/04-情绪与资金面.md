## 一、市场情绪指标体系

### 1.1 官方与市场情绪指标（A股特色）

| 指标 | 数据来源 | 计算逻辑 | 信号阈值 | 说明 |
|------|---------|---------|---------|------|
| **涨跌停比** | 交易所 | 涨停数 ÷ 跌停数 | >3 乐观；<0.5 悲观 | 反映短线情绪强度 |
| **连板高度** | 同花顺/东方财富 | 当日最高连板数 | ≥5 高热；≤2 低迷 | 衡量题材炒作烈度 |
| **炸板率** | 交易所 | 炸板数 ÷ (涨停数+炸板数) | >40% 情绪退潮 | 短线资金信心指标 |
| **昨日涨停今日表现** | 量化平台 | 涨停股次日平均涨跌幅 | >2% 情绪延续 | 检验赚钱效应持续性 |
| **两融余额** | 交易所 | 融资余额 - 融券余额 | 环比增2% 加杠杆 | 杠杆资金情绪风向标 |
| **新增开户数** | 中登公司 | 周度新开户数 | 环比增30% 散户入场 | 滞后指标，情绪见顶信号 |
| **中国波指（CVIX）** | 上交所 | 50ETF期权隐含波动率 | >25 恐慌；<15 乐观 | A股版"恐慌指数" |

### 1.2 情绪周期五阶段模型

| 阶段 | 量价特征 | 情绪指标表现 | 操作策略 |
|------|---------|-------------|---------|
| **低迷期** | 缩量阴跌，地量地价 | 涨停10家，跌停30家，CVIX>25 | 观望，筛选错杀标的 |
| **复苏期** | 温和放量，小阳线 | 涨停20-40家，连板高度回升 | 分批建仓优质标的 |
| **兴奋期** | 放量普涨，天量 | 涨停80家，炸板率20%，开户数暴增 | 持有为主，警惕高潮 |
| **怀疑期** | 滞涨放量，分化 | 涨停减少，高位股炸板增多 | 减仓高估值题材股 |
| **恐慌期** | 放量暴跌，千股跌停 | 跌停100家，CVIX>35 | 逆向布局，分批抄底 |

|---------|---------|---------|---------|---------|

### 2.2 核心资金指标计算

```python
import akshare as ak
import pandas as pd
import numpy as np

def get_north_capital(start_date='20240101'):
    """获取北向资金历史净流入"""
    try:
        df = ak.stock_hsgt_north_net_flow_in_em(symbol="沪股通")
        df = df[df['date'] >= start_date]
        df['cum_net_inflow'] = df['net_inflow'].cumsum()
        return df[['date', 'net_inflow', 'cum_net_inflow']]
    except Exception as e:
        print(f"数据获取失败: {e}")
        return pd.DataFrame()

def calculate_main_force_flow(df):
    """计算主力资金净流入（简化版）"""
    df = df.copy()
    df['avg_price'] = df['amount'] / (df['volume'] * 100 + 1e-6)
    df['is_large_order'] = (df['amount'] > 1_000_000).astype(int)  # 大单阈值100万
    df['direction'] = np.where(
        df['avg_price'] > df['close'].shift(1), 1,
        np.where(df['avg_price'] < df['close'].shift(1), -1, 0)
    )
    df['main_force_flow'] = df['amount'] * df['direction'] * df['is_large_order']
    return df.groupby('date')['main_force_flow'].sum().reset_index()
```

## 三、杠杆资金监控（融资融券）

### 3.1 核心指标

```python
def leverage_health_check(margin_balance, market_value, historical_avg):
    """杠杆健康度评估"""
    leverage_ratio = margin_balance / market_value * 100  # 杠杆率%
    deviation = (margin_balance - historical_avg) / historical_avg * 100
    
    if leverage_ratio < 2.5:
        risk_level = "低风险"
    elif leverage_ratio < 3.5:
        risk_level = "中风险"
    else:
        risk_level = "高风险"
    
    warning = "⚠️ 杠杆率3.5%需警惕系统性风险" if leverage_ratio >= 3.5 else None
    
    return {
        "杠杆率": round(leverage_ratio, 2),
        "偏离均值": round(deviation, 2),
        "风险等级": risk_level,
        "警告": warning
    }
```

## 四、情绪温度计（多维合成指标）

### 4.1 情绪评分模型（0~100分）

```python
def market_sentiment_score(limit_up_count, limit_down_count, cvix, turnover_rate, north_flow, new_accounts_ratio):
    """多维情绪评分（0-100分）"""
    
    # 1. 涨跌停情绪（0-30分）
    limit_ratio = limit_up_count / max(limit_up_count + limit_down_count, 1)
    score_limit = min(30, limit_ratio * 60)
    
    # 2. 波动率反向指标（0-25分）
    score_cvix = max(0, min(25, 25 - (cvix - 15) * 0.8))
    
    # 3. 换手活跃度（0-20分）
    score_turnover = min(20, turnover_rate * 6)
    
    # 4. 北向资金（0-15分）
    score_north = min(15, max(0, north_flow))
    
    # 5. 散户情绪（0-10分）
    score_account = min(10, max(0, new_accounts_ratio * 8))
    
    total_score = score_limit + score_cvix + score_turnover + score_north + score_account
    
    # 情绪等级判定
    if total_score <= 25:
        level, signal = "冰点", "机会"
    elif total_score <= 45:
        level, signal = "修复", "中性"
    elif total_score <= 70:
        level, signal = "健康", "中性"
    elif total_score <= 85:
        level, signal = "过热", "谨慎"
    else:
        level, signal = "沸腾", "高危"
    
    return {"评分": round(total_score, 1), "等级": level, "信号": signal}
```

### 4.2 情绪-仓位映射建议

| 情绪评分 | 市场状态 | 建议仓位 | 操作重点 |
|---------|---------|---------|---------|

## 七、风险控制与使用边界

⚠️ **本体系为辅助决策工具，不可单独作为交易依据**

| 风险类型 | 典型场景 | 应对措施 |
|---------|---------|---------|
| 情绪滞后 | 情绪高涨但政策突变（如2015年股灾） | 结合政策面交叉验证，设置硬止损 |
| 资金诱多 | 主力对倒制造流入假象（无真实涨幅） | 观察量价配合，避免纯看资金流 |
| 杠杆踩踏 | 融资盘集中平仓（2015年6月） | 避免高杠杆个股，总仓位≤70% |
| 极端行情 | 千股跌停/涨停（流动性枯竭） | 情绪指标失效，以风控为先，空仓观望 |
