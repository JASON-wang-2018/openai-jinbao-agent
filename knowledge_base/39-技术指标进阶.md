## 一、随机指标 (KDJ)

### 原理
通过研究最高价、最低价和收盘价之间的关系，判断股价的相对强弱和超买超卖状态。

### 组成
- **K线**：快速随机指标
- **D线**：慢速随机指标  
- **J线**：3×K - 2×D，敏感性最强

### 应用方法
1. **KD交叉信号**
   - K上穿D：金叉买入信号
   - K下穿D：死叉卖出信号

2. **J值判断**
   - J > 100：超买区域，随时准备卖出
   - J < 0：超卖区域，随时准备买入

3. **背离信号**
   - 顶背离：价格创新高，KDJ未创新高 → 卖出信号
   - 底背离：价格创新低，KDJ未创新低 → 买入信号

## 二、布林带指标 (BOLL)

### 原理
基于统计学的标准差计算，形成价格波动的通道。

### 组成
- **中轨**：20日简单移动平均线 (SMA)
- **上轨**：中轨 + 2倍标准差
- **下轨**：中轨 - 2倍标准差

### 应用方法
1. **突破交易**
   - 突破上轨：超买信号可能回落
   - 跌破下轨：超卖信号可能反弹

2. **收口信号**
   - 带口收缩：代表变盘在即，等待突破方向
   - 带口放大：趋势行情延续

3. **中轨支撑/压力**
   - 价格在中轨上方运行：中长期上升趋势
   - 价格在中轨下方运行：中长期下降趋势

## 三、能量潮指标 (OBV)

### 原理
累计成交量变化，判断资金流向和趋势可持续性。

### 计算方法
- 今日收盘价 > 昨日收盘价：OBV = 昨日OBV + 今日成交量
- 今日收盘价 < 昨日收盘价：OBV = 昨日OBV - 今日成交量
- 今日收盘价 = 昨日收盘价：OBV 不变

### 应用方法
1. **趋势判断**
   - OBV 上升趋势：资金持续流入，趋势可能延续
   - OBV 下降趋势：资金持续流出，趋势可能终结

2. **背离信号**
   - OBV 向上，股价向下：底背离 → 潜在上涨信号
   - OBV 向下，股价向上：顶背离 → 潜在下跌信号

3. **横盘突破**
   - OBV 长期横盘后向上突破：买入信号
   - OBV 长期横盘后向下突破：卖出信号

## 四、平均真实波幅 (ATR)

### 原理
衡量股价在一定周期内的平均波动幅度，帮助设置止损和判断市场活跃度。

### 计算方法
TR（真实波幅）= max(H-L, |H-PC|, |L-PC|)
- H：当日最高价
- L：当日最低价
- PC：前一日收盘价

ATR = TR 的 N 日简单移动平均（通常 N=14）

### 应用方法
1. **止损设置**
   - 激进止损：价格 - 1.5 × ATR
   - 保守止损：价格 - 2.5 × ATR

2. **市场活跃度**
   - ATR 值大：市场波动剧烈，风险较高
   - ATR 值小：市场波动平缓，风险较低

3. **仓位管理**
   - 高 ATR → 降低仓位
   - 低 ATR → 提高仓位

## 五、实战组合应用

### 趋势确认组合
```
布林带 + ATR + MA60
- 布林带中轨向上 + ATR 放大 + 价格站上 MA60 → 上升趋势确认
- 布林带中轨向下 + ATR 放大 + 价格跌破 MA60 → 下降趋势确认
```

### 买卖点组合
```
KDJ + OBV + 成交量
- KDJ 金叉 + OBV 上升 + 成交量放大 → 买入信号
- KDJ 死叉 + OBV 下降 + 成交量萎缩 → 卖出信号
```

### 风险控制组合
```
ATR + 布林带
- 止损：价格 - 2 × ATR
- 止盈：价格 + 3 × ATR 或 跌破布林带中轨
```
