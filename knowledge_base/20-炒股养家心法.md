## 二、养家战法核心

### 2.1 龙头战法

```
核心思想：
├── 只做市场总龙头
├── 只做板块龙头
├── 只做情绪转折点的龙头
└── 龙头倒下，行情结束

操作要点：
1. 竞价确认板块强度
2. 盘中确认龙头地位
3. 放量烂板不跑，缩量涨停持有
4. 收盘不板，次日跑路
```

### 2.2 情绪周期战法

```
冰点 → 复苏 → 发酵 → 高潮 → 分化 → 冰点

操作策略：
├── 冰点：空仓等待，不轻易抄底
├── 复苏：试探性建仓，试错
├── 发酵：加仓，紧跟主线
├── 高潮：持股不动，边打边撤
├── 分化：去弱留强，锁定利润
└── 冰点：空仓等待
```

### 2.3 追涨战法

```
适用场景：
├── 市场情绪好
├── 板块效应强
├── 龙头刚启动

买入条件：
├── 9:25竞价涨幅 > 3%
├── 板块涨幅 > 大盘
├── 板块涨停家数 > 3
└── 个股承接强

止损纪律：
├── 开盘不及预期，-3%止损
├── 收盘不板，次日-3%止损
└── 跌破5日线，止损
```

### 2.4 低吸战法

```
适用场景：
├── 市场震荡
├── 龙头回调
├── 情绪退潮期

低吸条件：
├── 龙头首阴
├── 回调至关键支撑（MA5/MA10）
├── 成交量萎缩50%以上
└── 分时企稳

止损纪律：
├── 跌破支撑，止损
├── 亏损超过5%，止损
└── 3天内不涨，止损
```

## 三、养家操作纪律

### 3.1 买入纪律

```
1. 只买龙头，不买杂毛
2. 计划好再买，不临时起意
3. 分批建仓，不一次性满仓
4. 竞价确认，盘中验证，收盘确认
5. 不追高位缩量板
6. 不做跟风股
7. 不做首板以下的股票
```

### 3.2 卖出纪律

```
1. 不及预期，果断卖出
2. 收盘不板，次日竞价跑
3. 龙头首阴，次日不反包跑
4. 板块退潮，果断清仓
5. 跌破止损位，无条件止损
6. 盈利超过30%，分批减仓
7. 不意淫反弹，不幻想行情
```

### 3.3 仓位管理

```
总仓位控制：
├── 单只个股 ≤ 30%
├── 单个板块 ≤ 50%
├── 总仓位根据行情调整
│   ├── 主升行情：80-100%
│   ├── 震荡行情：50-80%
│   ├── 退潮行情：0-30%
│   └── 冰点行情：0%
└── 新题材试错：≤ 20%

加减仓规则：
├── 盈利加仓：浮盈>10%后加仓
├── 亏损不加仓：亏损不加仓
└── 分批建仓：首仓→确认→加仓
```

## 四、养家心法详解

### 4.1 关于选股

```
选股原则：
1. 只选热点板块
2. 只选板块龙头
3. 只选成交额前100
4. 只选股性活的股票
5. 只选有人气的股票

排除法：
✗ 排除ST
✗ 排除亏损股
✗ 排除成交额<1亿
✗ 排除基本面有雷
✗ 排除流动性差
```

### 4.2 关于持股

```
持股心态：
1. 买入后不盯盘
2. 相信自己的判断
3. 不受外界干扰
4. 按计划执行

持股条件：
├── 板块不退潮
├── 龙头不倒
├── 承接有力
├── 量能不异常
└── 浮盈足够安全垫

持股技巧：
├── 涨停板持有
├── 烂板看收盘
├── 炸板看回封
└── 尾盘不板跑
```

### 4.3 关于空仓

```
空仓时机：
1. 行情不好
2. 没有机会
3. 连续亏损
4. 看不懂行情
5. 情绪不稳定

空仓好处：
1. 避免亏损
2. 保存实力
3. 等待机会
4. 心态平和
5. 机会来了有子弹

空仓不是逃避，而是等待
```

## 十、实战练习

### 练习题1

```
题目：今日大盘情绪冰点，板块普跌，
      午后有一只股票逆势涨停，
      板块内仅此一只涨停，
      该如何操作？

答案：
1. 先观察，不急于买入
2. 看次日竞价是否超预期
3. 次日竞价>3%可轻仓试错
4. 仓位控制在20%以内
5. 止损位：-5%
```

### 练习题2

```
题目：龙头股连续涨停5天后，
      今日高位放量烂板，
      板块内跟风股大面积炸板，
      该如何操作？

答案：
1. 减仓一半
2. 收盘不板清仓
3. 次日不反包彻底清仓
4. 不抄底
5. 等待新周期
```
