## 一、核心认知：股票是周期性游戏

### 1.1 市场周期四阶段

```
┌─────────────────────────────────────────────────────────────┐
│                    市场情绪周期                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  冰点 → 复苏 → 主升 → 高潮 → 衰退 → 冰点                  │
│                                                             │
│  • 冰点：无人问津，成交量萎缩                             │
│  • 复苏：试探性上涨，板块轮动                             │
│  • 主升：趋势形成，赚钱效应扩散                          │
│  • 高潮：人人都在谈论股票，估值泡沫                       │
│  • 衰退：趋势反转，跌幅扩大                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 情绪周期对应策略

| 阶段 | 特征 | 策略 |
|------|------|------|
| 冰点 | 涨停少，情绪低迷 | 轻仓试错，选股观察 |
| 复苏 | 板块轮动，涨停增多 | 逐步加仓，跟随主线 |
| 主升 | 趋势确立，加速上涨 | 持股待涨，让利润奔跑 |
| 高潮 | 狂热，二八分化 | 逐步减仓，止盈 |
| 衰退 | 下跌趋势形成 | 空仓等待 |

## 二、主力行为识别

### 2.1 主力 vs 庄家

| | 主力 | 庄家 |
|--|------|------|

| **风险** | 顺势而为 | 流动性风险 |

### 2.2 主力分类

| 类型 | 资金规模 | 周期 | 特点 |
|------|---------|------|------|
| 短线主力 | 几千万-1亿 | 1-3个月 | 快进快出，题材炒作 |
| 中线主力 | 1-5亿 | 3-6个月 | 板块行情 |

## 三、主力坐庄七步法

```
选股 → 建仓 → 洗盘 → 试盘 → 拉升 → 出货 → 砸盘
```

### 3.1 建仓阶段（3-6个月）

**目的**：收集足够筹码

**特征**：
- 成交量：温和放大
- 股价：涨跌无量
- K线：小阴小阳，振幅收窄
- 均线：开始走平

**建仓方式**：
| 方式 | 特征 | 识别 |
|------|------|------|
| 打压建仓 | 快速砸盘，制造恐慌 | 利空配合，连续大跌 |
| 拉高建仓 | 快速脱离成本区 | 利好配合，持续放量 |
| 横盘建仓 | 长期横盘震荡 | 量能极度萎缩 |

### 3.2 洗盘阶段（1-3个月）

**目的**：
1. 清理浮筹
2. 抬高散户持仓成本
3. 试盘测试跟风盘强弱

**四大洗盘战术**：

| 战术 | 核心逻辑 | 散户心理 |
|------|---------|---------|
| **恐慌式打压** | 制造短期剧烈亏损 | 恐惧、止损本能 |
| **迷惑式震荡** | 制造无序波动 | 迷惑、贪婪与恐惧 |
| **窒息式盘整** | 漫长时间消磨耐心 | 焦虑、厌倦 |
| **欺诈式挂单** | 盘口制造虚假信号 | 从众、盘口依赖 |

**16种洗盘手法详解**：

1. **打压洗盘**：开盘快速放量下砸，关键位神秘止跌
2. **破位洗盘**：故意击穿技术支撑位，引发止损
3. **跌停挂出法**：挂巨额卖单制造绝望感，但不真砸
4. **长阴砸盘**：连续大阴线制造恐慌
5. **上冲下洗**：宽幅震荡，高抛低吸
6. **假突破**：突破后快速回落
7. **横盘整理**：长时间横盘消磨耐心
8. **串阴洗盘**：连续小阴线回调
9. **缩量回调**：下跌缩量，是洗盘不是出货
10. **尾盘打压**：收盘前快速下砸
11. **早盘冲高**：高开后回落
12. **大单压盘**：卖盘挂大单制造压力
13. **大单托盘**：买盘挂大单制造支撑假象
14. **对倒拉升**：自买自卖制造放量
15. **委托买单**：虚假买单诱导
16. **虚假撤单**：挂单又撤制造假象

### 3.3 拉升阶段（1-3个月）

**特征**：
- 成交量：持续放量
- 股价：沿5日均线上涨
- 角度：陡峭，45度以上
- 涨停增多：连续涨停板

**散户心理**：从怀疑到追涨

### 3.4 出货阶段（1-2个月）

**特征**：
- 成交量：放量滞涨
- 股价：高位震荡
- 利好频出：配合减持
- 涨少跌多

**出货方式**：
1. **高位横盘出货**：高位长期横盘
2. **涨停出货**：板上出货
3. **冲高出货**：大幅高开后回落
4. **利好出货**：配合利好消息

## 四、交易心理与人性弱点

### 4.1 七宗罪

| 弱点 | 表现 | 后果 |
|------|------|------|
| **贪婪** | 赚还想赚更多 | 盈利变亏损 |
| **恐惧** | 跌就恐慌卖 | 底部割肉 |
| **侥幸** | 套牢等解套 | 亏损扩大 |
| **懒惰** | 不学习复盘 | 重复犯错 |
| **骄傲** | 赚了就嘚瑟 | 盲目自信 |
| **嫉妒** | 别人赚了自己亏 | 追涨杀跌 |
| **愤怒** | 亏了就发怒 | 情绪化交易 |

### 4.2 心理误区

**损失厌恶**：
- 亏钱痛苦程度 ≈ 赚钱快乐程度的2倍
- 亏损时不愿止损
- 盈利时过早卖出

**过度自信**：
- 连续盈利后加大仓位
- 认为自己能预测市场

**确认偏误**：
- 只看支持自己观点的信息
- 忽视不利信号

### 4.3 主力利用的心理

| 心理弱点 | 主力利用方式 |
|---------|-------------|
| 恐惧 | 打压股价制造恐慌 |
| 贪婪 | 拉升吸引追涨 |
| 从众 | 制造放量 侥幸上涨假象 |
| | 下跌时散户等待解套 |
| 损失厌恶 | 频繁小额亏损让散户不愿割肉 |

## 五、识别主力行为的实战技巧

### 5.1 成交量是关键

```
量在价先 —— 成交量是主力行为的痕迹
```

- 建仓：缩量横盘
- 拉升：放量上涨
- 出货：放量滞涨
- 洗盘：缩量回调

### 5.2 盘口语言

| 信号 | 含义 |
|------|------|
| 卖一挂大单 | 压制股价 |
| 买一挂大单 | 支撑股价 |
| 上下对倒 | 自买自卖制造放量 |
| 尾盘拉升 | 吸引次日跟风 |
| 尾盘打压 | 第二天低开吸筹 |

### 5.3 分时图特征

| 形态 | 含义 |
|------|------|
| 早盘拉高后横盘 | 主力控盘能力强 |
| 尾盘急拉 | 短线行为 |
| 盘中急跌后拉回 | 洗盘 |
| 波浪式上涨 | 稳步拉升 |

## 六、总结：散户的应对策略

### 6.1 核心原则

1. **看量不看价**：成交量反映主力意图
2. **顺势而为**：不做逆势
3. **截断亏损**：让利润奔跑
4. **独立思考**：不盲目从众

### 6.2 识别主力痕迹

- 缩量横盘 → 主力建仓
- 放量突破 → 主力启动
- 缩量回调 → 主力洗盘
- 放量滞涨 → 主力出货

### 6.3 心理建设

- 接受亏损是交易的一部分
- 不要试图买在最低卖在最高
- 等待确定性机会
- 只做看得懂的行情

---

*记住：股市是零和博弈，主力的利润来自散户的亏损。理解主力行为，就是保护自己。*
