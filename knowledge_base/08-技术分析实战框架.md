## 三、第1步：定性定位 —— 趋势与生命周期判断

**目标**：先判断"这只股票现在处在哪一阶段"，避免在错误周期内做精细分析。

### 3.1 核心判断维度

| 维度 | 判断内容 | 信号说明 |
|------|---------|---------|
| **均线结构** | 多头/空头/震荡 | 5/10/20/60日线排列 |
| **股价位置** | 历史低位/中位/高位 | 相对历史区间位置 |
| **趋势属性** | 主升/震荡/下跌反弹/出清末期 | 当前运行阶段 |

### 3.2 均线结构判断

```
多头排列：   MA5 > MA10 > MA20 > MA60 → 强势，可操作
空头排列：   MA5 < MA10 < MA20 < MA60 → 弱势，观望为主
震荡整理：   均线缠绕，方向不明 → 等待方向选择
```

### 3.3 股价位置判断

| 位置 | 特征 | 操作意义 |
|------|------|---------|
| **低位** | 距历史高点跌幅>50% | 吸筹可能，关注 |
| **中位** | 距高低点中间 | 换手、洗盘 |
| **高位** | 距历史高点<20% | 派发风险，谨慎 |

### 3.4 生命周期阶段

```
┌────────────────────────────────────────────────────────────┐
│  吸筹建仓 → 洗盘整理 → 拉升启动 → 主升浪 → 高位震荡 → 派发出货 → 下跌出清  │
│     ↓          ↓           ↓          ↓          ↓           ↓           ↓
│   3-6月      1-3月       几天        1-3月      1-2月       1-2月       1-3月
│                                                                           
│  【操作价值】                                                           
│     高          高          高          高         中          低          低
└────────────────────────────────────────────────────────────┘
```

**输出结论**：
> 该股票目前处于【XX阶段】，均线结构【多/空/震】，量价关系【健康/异常】，主力行为偏向【XX】。

## 四、第2步：多维分析 —— 证据收集而非单点判断

**目标**：避免"单一信号误判"，通过多维一致性判断主力真实意图。

### 4.1 趋势与均线分析

**关键信号**：
- ✅ 均线被有效回踩而不破 → 强势信号
- ✅ 回踩缩量 → 健康洗盘
- ✅ 均线斜率保持上行 → 趋势延续

**量化指标**：
```python
def 均线结构得分(df):
    ma5, ma10, ma20, ma60 = 计算均线(df)
    
    if ma5 > ma10 > ma20 > ma60:
        return 20  # 多头排列
    elif ma5 < ma10 < ma20 < ma60:
        return 0   # 空头排列
    else:
        return 10  # 震荡整理
```

### 4.2 K线形态与组合

| 形态名称 | K线特征 | 市场含义 |
|---------|---------|---------|
| **试盘线** | 长上影、缩量 | 测试上方抛压 |
| **洗盘线** | 假阴真阳 | 清洗浮筹 |
| **启动线** | 放量中阳/长阳 | 开始拉升 |
| **仙人指路** | 长上影线 | 试探压力 |
| **早晨之星** | 两阴夹一阳 | 底部反转 |
| **黄昏之星** | 两阳夹一阴 | 顶部反转 |

**关键点**：关键位置出现连续K线确认

### 4.3 量价关系（核心）

| 量价形态 | 信号强度 | 含义 |
|---------|---------|------|
| 价涨量增 | 强 ✅ | 健康上涨 |
| 价涨量缩 | 中 ⚠️ | 虚涨诱多 |
| 价跌量增 | 强 ✅ | 恐慌见底 |
| 价跌量缩 | 弱 ❌ | 阴跌不止 |
| 放量长阴不破位 | 强 ✅ | 主力洗盘 |
| 放量不涨 | 弱 ❌ | 主力出货 |

**核心原则**：
- 上涨必须放量
- 回调必须缩量
- 量价配合 = 主力做多

### 4.4 分价表 / 成交结构

**分析要点**：
- 成交密集区是否上移
- 是否存在明显"锁仓成本区"
- 高位成交是否快速被吃掉

**判断逻辑**：
```python
def 分析分价表(分价数据):
    密集区 = 找出成交密集区间()
    锁仓区 = 识别锁仓成本区()
    套牢盘 = 分析高位套牢分布()
    
    return {
        '密集区位置': 密集区,
        '锁仓区': 锁仓区,
        '套牢盘分布': 套牢盘
    }
```

### 4.5 动能与节奏

**主力常见操盘节奏**：

| 阶段 | 操盘节奏 | 散户应对 |
|------|---------|---------|
| 吸筹 | 温和放量，震荡上行 | 分批建仓 |
| 洗盘 | 缩量回调，不破关键位 | 持股不动 |
| 拉升 | 放量突破，流畅上涨 | 顺势持有 |
| 派发 | 宽幅震荡，量能放大 | 分批减仓 |

### 4.6 板块与题材环境

| 检查项 | 利好信号 | 利空信号 |
|-------|---------|---------|
| 板块当日表现 | 领涨 + 放量 | 跟跌 + 缩量 |
| 板块近期持续性 | 连涨3天+ | 连跌3天+ |
| 政策/事件驱动 | 有利好催化 | 无利好支撑 |
| 个股板块地位 | 龙头/领涨 | 跟风/补涨 |

### 4.7 主力资金行为

| 行为类型 | 特征 | 判断 |
|---------|------|------|
| **推升** | 持续净流入、量价配合 | 真正做多 |
| **托而不攻** | 护盘但不拉涨 | 维持出货 |
| **对倒** | 量能放大、价格不涨 | 主力内耗 |
| **洗盘** | 急跌后快速拉回 | 清洗浮筹 |

## 五、第3步：综合评分 —— 量化主观判断

**目标**：把"感觉"转化为"可比较的分数"。

### 5.1 评分模型（100分制）

| 分析维度 | 分值 | 评分标准 |
|---------|------|---------|
| **趋势结构** | 20 | 多头=20, 震荡=10, 空头=0 |
| **量价健康度** | 25 | 价涨量增=25, 异常=0 |
| **K线信号质量** | 15 | 启动形态=15, 无信号=5 |
| **主力行为可信度** | 20 | 一致性高=20, 混乱=5 |
| **板块与情绪环境** | 20 | 热点+领涨=20, 冷门=5 |

### 5.2 评分结果解读

| 分数区间 | 等级 | 操作建议 |
|---------|------|---------|
| **80+** | 强势主升 | 积极参与，趋势持有 |
| **60-80** | 可操作 | 回调买入，择时操作 |
| **40-60** | 观望 | 等信号，不追高 |
| **40以下** | 风险区 | 回避或仅短线 |

### 5.3 评分计算示例

```
股票A评分：
├─ 均线结构：18/20 (多头排列，略有回踩)
├─ 量价关系：22/25 (上涨放量，回调缩量)
├─ K线形态：12/15 (出现启动线)
├─ 主力行为：16/20 (推升行为一致)
└─ 板块环境：15/20 (板块跟风，强度中等)
    
总分：83分 → 强势主升结构 → 积极参与
```

## 六、第4步：走势推演 —— 概率化而非预测

**目标**：提前准备应对方案，而不是"赌方向"。

### 6.1 三种路径推演

| 路径 | 触发条件 | 概率 | 应对策略 |
|------|---------|------|---------|
| **主升延续** | 缩量回踩不破关键均线 | 50% | 持有或加仓 |
| **震荡洗盘** | 放量不涨、反复假突破 | 35% | 减仓观望 |
| **趋势破坏** | 放量跌破关键结构位 | 15% | 止损离场 |

### 6.2 关键位识别

```python
def 识别关键位(df):
    """识别关键支撑/压力位"""
    前高 = df['high'].rolling(20).max().iloc[-1]
    前低 = df['low'].rolling(20).min().iloc[-1]
    ma20 = df['close'].rolling(20).mean().iloc[-1]
    ma60 = df['close'].rolling(60).mean().iloc[-1]
    
    return {
        '压力位': 前高,
        '支撑位': max(前低, ma20),
        '强支撑': ma60
    }
```

### 6.3 概率化表达

```
后续走势概率分布：
├─ 主升延续（55%）：缩量回踩MA10后企稳，继续上行
├─ 震荡洗盘（30%）：在MA20-MA60区间宽幅震荡
└─ 趋势破坏（15%）：放量跌破MA20，向MA60寻求支撑
```

## 七、第5步：制定策略 —— 按"持仓状态"区分

**目标**：不同人，不同策略，避免一刀切。

### 7.1 持仓状态策略

| 持仓状态 | 策略要点 |
|---------|---------|
| **空仓者** | 等确认信号（放量突破 / 回踩确认） |
| **轻仓者** | 回踩关键位可加仓，不追高 |
| **重仓者** | 以风控为第一优先级，不加仓，只做减法 |

### 7.2 具体操作建议

**空仓者**：
- 等待放量突破关键压力位后介入
- 或等待回踩关键支撑确认后介入
- 仓位：首次30%，确认后加至50%

**轻仓者**：
- 回踩MA10/MA20企稳可加仓
- 加仓后持仓不超过70%
- 不追高，不参与缩量上涨

**重仓者**：
- 以持有为主，不加仓
- 设置移动止盈保护利润
- 出现出货信号分批减仓

## 八、第6步：设定风控 —— 系统性活下来

### 8.1 止损规则

| 止损类型 | 设置方法 | 适用场景 |
|---------|---------|---------|
| **结构止损** | 跌破关键结构位 | 趋势交易 |
| **均线止损** | 跌破重要均线 | 短线交易 |
| **成本止损** | 跌破成本密集区下沿 | 抄底交易 |
| **时间止损** | 持仓N天不涨 | 突破交易 |

### 8.2 止盈策略

```
分批止盈方案（以20%涨幅为例）：
├── 涨10% → 止盈1/3（锁定33%利润）
├── 涨15% → 止盈1/3（累计锁定55%利润）
├── 涨20% → 止盈剩余（目标达成）
└── 跌破5日均线 → 清仓（保护剩余利润）
```

### 8.3 风控观察点

| 观察点 | 信号强度 | 应对 |
|--------|---------|------|
| **放量不涨** | 强 ⚠️ | 主力出货信号，减仓 |
| **板块退潮** | 中 ⚠️ | 跟风股风险，减仓 |
| **主力护盘消失** | 强 ⚠️ | 趋势终结，止损 |
| **跌破关键位** | 强 🔴 | 纪律性止损 |
