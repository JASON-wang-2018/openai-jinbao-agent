# 风险管理

## 一、仓位管理

### 1.1 仓位控制原则

| 市场状态 | 建议仓位 | 理由 |
|---------|---------|------|
| 牛市主升 | 80-100% | 顺势而为 |
| 震荡市 | 50-70% | 低吸高抛 |
| 熊市/风险期 | 0-30% | 现金为王 |
| 不确定性高 | 30-50% | 轻仓观望 |

### 1.2 仓位分配公式

#### 金字塔加仓
```
第1次买入：40%  ← 确定性最高时
第2次买入：30%  ← 回调企稳后
第3次买入：20%  ← 突破确认后
第4次买入：10%  ← 追涨确认后
```

#### 倒金字塔减仓
```
第1次减仓：10%  ← 盈利30%后
第2次减仓：20%  ← 盈利50%后
第3次减仓：30%  ← 盈利80%后
第4次减仓：40%  ← 目标达成
```

### 1.3 单股仓位限制

| 账户规模 | 单股最大仓位 | 分股数量 |
|---------|-------------|---------|
| <10万 | 100% | 1-2只 |
| 10-50万 | 50% | 3-5只 |
| 50-100万 | 30% | 5-8只 |
| >100万 | 20% | 8-10只 |

## 二、止损策略

### 2.1 固定止损

| 类型 | 设置方法 | 适用场景 |
|------|---------|---------|
| 固定金额 | 亏损达到X元止损 | 短线交易 |
| 固定比例 | 亏损达到X%止损 | 趋势交易 |
| 技术止损 | 跌破支撑位止损 | 技术分析 |

### 2.2 动态止损

| 类型 | 设置方法 | 优点 |
|------|---------|------|
| 移动止损 | 随盈利上移止损点 | 保护利润 |
| ATR止损 | 基于波动率设置 | 适应市场 |
| 时间止损 | 持仓X天不涨止损 | 避免套牢 |

### 2.3 止损比例参考

| 交易风格 | 建议止损 | 交易周期 |
|---------|---------|---------|
| 超短线 | 3-5% | 日内 |
| 短线 | 5-10% | 1-5天 |
| 中线 | 10-20% | 1-4周 |
| 长线 | 20-30% | 1-6月 |

## 三、风险指标

### 3.1 个股风险指标

| 指标 | 计算方法 | 风险提示 |
|------|---------|---------|
| 最大回撤 | 峰值到谷值的最大跌幅 | 历史最大亏损 |
| VaR（风险价值） | 95%置信度下的最大损失 | 未来可能损失 |
| 波动率 | 收益率标准差 | 价格波动程度 |
| beta | 与大盘的相关系数 | 系统性风险 |

### 3.2 组合风险指标

| 指标 | 计算方法 | 优化目标 |
|------|---------|---------|
| 夏普比率 | (收益-无风险)/波动率 | 风险调整收益 |
| 索提诺比率 | (收益-无风险)/下行波动率 | 下行风险收益 |
| 卡玛比率 | 收益/最大回撤 | 性价比 |
| 信息比率 | 超额收益/跟踪误差 | 主动管理能力 |

## 四、分散投资

### 4.1 分散维度

| 维度 | 方法 | 效果 |
|------|------|------|
| 行业分散 | 不同行业配置 | 行业风险 |
| 风格分散 | 成长+价值 | 风格轮动 |
| 周期分散 | 周期+防御 | 经济周期 |
| 时间分散 | 分批建仓 | 入场时机 |

### 4.2 相关性配置

| 相关性 | 配置建议 |
|--------|---------|
| 高相关（>0.7） | 减少重复配置 |
| 中相关（0.3-0.7） | 适量配置 |
| 低相关（<0.3） | 优先配置 |

## 五、风险控制体系

### 5.1 事前控制

1. **仓位限制**：单股不超X%，总仓位Y%
2. **分散度**：至少N只不同行业
3. **标的选择**：避开ST、问题股、高质押
4. **流动性**：日成交额>5000万

### 5.2 事中控制

1. **动态监控**：每日检查持仓风险
2. **预警机制**：触发止损/风控线提醒
3. **纪律执行**：到达止损点坚决执行

### 5.3 事后控制

1. **复盘分析**：为什么亏损
2. **教训总结**：改进交易系统
3. **持续优化**：不断完善风控体系

## 六、常见风险类型

### 6.1 系统性风险
- **表现**：大盘暴跌，所有股下跌
- **应对**：降低仓位，持有现金

### 6.2 行业风险
- **表现**：政策变化，行业利空
- **应对**：分散行业，规避高危行业

### 6.3 个股风险
- **表现**：个股黑天鹅，业绩暴雷
- **应对**：分散持仓，设止损，避开问题股

### 6.4 流动性风险
- **表现**：想卖卖不掉，跌停板
- **应对**：关注成交量，不碰仙股

### 6.5 杠杆风险
- **表现**：爆仓，倒欠券商钱
- **应对**：控制杠杆率，留足安全边际
