## 📊 常见场景示例

### 场景1: 分析销售数据
```python
from excel_processor import ExcelProcessor

processor = ExcelProcessor()

# 读取销售数据
df = processor.read_excel('销售数据.xlsx')

# 按区域统计
print("\n各区域销售:")
print(df.groupby('区域')['销售额'].sum())

# 查看TOP 5产品
print("\nTOP 5产品:")
print(df.nlargest(5, '销售额')[['产品', '销售额']])
```

### 场景2: 数据清洗
```python
from excel_processor import ExcelProcessor

processor = ExcelProcessor()

# 读取原始数据
df = processor.read_excel('原始数据.xlsx')

# 清洗数据
df_clean = processor.clean_data(
    df,
    dropna_cols=['姓名'],           # 删除姓名为空的行
    fillna_dict={'年龄': 30},       # 填充年龄空值
    strip_cols=['姓名', '城市']     # 去除空格
)

# 保存清洗后的数据
processor.write_excel(df_clean, '清洗后数据.xlsx')
processor.format_excel('清洗后数据.xlsx')
```

### 场景5: 创建透视表
```python
from excel_processor import ExcelProcessor

processor = ExcelProcessor()
df = processor.read_excel('销售数据.xlsx')

# 创建透视表：产品 × 季度
pivot = processor.create_pivot_table(
    df,
    index='产品',
    columns='季度',
    values='销售额',
    aggfunc='sum'
)

# 保存透视表
pivot.to_excel('销售透视表.xlsx')
processor.format_excel('销售透视表.xlsx')
```

```python
    '开始日期': ['2024-01-16', '2024-01-16'],
    '计划完成': ['2024-01-20', '2024-01-25'],

```

## 💡 常用技巧

### 数据筛选
```python
# 单条件
df[df['销售额'] > 100000]

# 多条件
df[(df['区域'] == '北京') & (df['销售额'] > 100000)]

# 包含筛选
df[df['产品'].isin(['产品A', '产品B'])]
```

### 数据排序
```python
# 单列排序
df.sort_values('销售额', ascending=False)

# 多列排序
df.sort_values(['区域', '销售额'], ascending=[True, False])
```

### 分组统计
```python
# 求和
df.groupby('区域')['销售额'].sum()

# 平均
df.groupby('区域')['销售额'].mean()

# 计数
df.groupby('状态').size()

# 多统计量
df.groupby('区域')['销售额'].agg(['sum', 'mean', 'count'])
```

### 数据清洗
```python
# 删除空值
df.dropna()

# 填充空值
df.fillna(0)

# 删除重复
df.drop_duplicates()

# 数据类型转换
df['日期'] = pd.to_datetime(df['日期'])
df['数值'] = pd.to_numeric(df['数值'], errors='coerce')
```

### 练习3: 处理销售数据
```python
import pandas as pd
from excel_processor import ExcelProcessor

processor = ExcelProcessor()

# 假设你有销售数据
df = pd.DataFrame({
    '产品': ['A', 'B', 'C', 'A', 'B', 'C'],
    '区域': ['北京', '上海', '广州', '北京', '上海', '广州'],
    '季度': ['Q1', 'Q1', 'Q1', 'Q2', 'Q2', 'Q2'],
    '销售额': [100, 150, 200, 120, 180, 220]
})

# 完成以下任务:
# 1. 计算每个区域的总销售额
# 2. 计算每个季度的平均销售额
# 3. 找出销售额最高的产品
# 4. 创建透视表并保存

# 答案:
print("\n各区域销售:")
print(df.groupby('区域')['销售额'].sum())

print("\n各季度平均:")
print(df.groupby('季度')['销售额'].mean())

print("\n最高销售额产品:")
print(df.loc[df['销售额'].idxmax(), '产品'])

pivot = processor.create_pivot_table(df, '产品', '季度', '销售额', 'sum')
processor.write_excel(pivot, '销售分析.xlsx')
```

## ❓ 常见问题

**Q: 如何读取多个 sheet？**
```python
sheet_names = processor.get_sheet_names('data.xlsx')
for name in sheet_names:
    df = processor.read_excel('data.xlsx', sheet_name=name)
    print(f"\n{name}:")
    print(df.head())
```

**Q: 如何处理大文件？**
```python
# 分块读取
for chunk in pd.read_excel('large.xlsx', chunksize=1000):
    process(chunk)
```

**Q: 如何设置 Excel 列宽？**
```python
# format_excel 会自动调整列宽
processor.format_excel('output.xlsx')
```

**Q: 如何批量处理多个文件？**
```python
import glob

for file in glob.glob('data/*.xlsx'):
    df = processor.read_excel(file)
    # 处理数据
    processed = process_data(df)
    processor.write_excel(processed, f'processed/{file}')
```
